　　第603章
<br/>　　在下定决心后，金问道双腿一夹座下马，挺长枪攻向野原火。这一枪来势汹汹，足见他在枪法上的造诣不浅。
<br/>　　野原火早已等待他出手，当枪扎过来时，他的刀也扬了起来。刀与枪战在一起，捉对厮杀，两匹马风车一般旋转，打得难解难分。
<br/>　　金问道的功夫本身就不弱，不然也不可能成为北蛮部的九王子。当初的北蛮王选择王子，每一人都有出色的本领。只不过，金问道已有许久没摸枪了，技艺上难免有些生疏。
<br/>　　双方将士都看呆了，每颗心都提了起来。场上二人的较量，已关乎这场战斗的胜负。任谁也没想到，金问道和野原火都有如此惊人的马上武艺。
<br/>　　金问道是如今的北蛮王，生来与枪棒为伴，武艺出众倒也不值得令人大惊小怪。只不过这野原火究竟是从哪里冒出来的，一身武艺又从何处学来，这令所有人都想不通。
<br/>　　远远的山坡上，立着两道人影。若有认得的，怕是会大吃一惊。王城大长老竟然亲自现身此地，此刻他裹了一件宽大的袍子，身旁立着柳褐岩。
<br/>　　柳褐岩最近的日子过得提心吊胆，燕不归回来了，还点名要他的性命。怎么想，牺牲自己也比牺牲燕不归更值得。如此，他只能拼命表现，希望大长老能法外开恩，给自己留一条活路。
<br/>　　金问道向呼延力求援，合两家之力消灭野原火。呼延力的确派了人来，还是由大长老亲自领军，足足三千健壮儿郎，此刻就立在大长老身后。
<br/>　　这三千人投入战场，已经可以左右这场战斗的胜败。
<br/>　　柳褐岩远远看着，道：“大长老，是不是要下令进攻？”
<br/>　　“等一等。”白无双缓缓摇头，目光看着远处两军对阵处，道：“哪个是野原火，你指给我看？”
<br/>　　“看情形与金问道一起厮杀的，那个独眼汉子就是野原火。”
<br/>　　“就是他么……”
<br/>　　白无双轻轻道了一声，目光落在了野原火身上。看年纪并不大，但动作却很干净，并没有初学者那种拖泥带水，仔细观察，也并非仗着力大伤人，年纪轻轻竟然有相当不错的武道造诣。
<br/>　　柳褐岩搞不懂这位大长老心里想什么，也不敢揣测，只是奉承道：“大长老见多识广，依长老的见识，这二人谁能取胜？”
<br/>　　白无双伸出手来，远远一指野原火，斩钉截铁道：“此人若非悟性惊人，便必然有名师传授。”
<br/>　　柳褐岩道：“那么，谁会获胜？”
<br/>　　白无双连看都懒得看他一眼，很明显，柳褐岩的悟性不如何，在武道上不可能有太大进步了。
<br/>　　两人的决斗，白无双为何单单提野原火，而不提金问道。他心中更看好谁，岂不是很明显。不说二人在武道上的差距，单单看二人表现出来的气势就可看出大概。
<br/>　　金问道已是戎族的北蛮王，主宰数万人的生死，这已令他满足。他现在的目的，是守住自己所拥有的。而野原火定然更贪婪，他不满足现状，想要的更多。一个守成，一个贪婪，胜负其实已一目了然。
<br/>　　而现在白无双关心的是，这野原火究竟有多贪婪，他够不够贪婪？
<br/>　　白无双不愧是宗师级别的人物，眼光还是有的。金问道的招式如他所料，先保证自己立于不败之地，再寻机进攻。而野原火则完全是不要命的打法，一心一意取对方性命，为了达到这个目的，便是杀敌一千，自损八百也在所不惜。
<br/>　　本来双方的差距并不明显，但在如此气势下，金问道则是越打越怯，最后铛地一声，野原火的刀背砸在铁枪上。金问道虎口一震，铁枪竟险些脱手掉在地上。
<br/>　　仗着一招之便，野原火已经要取金问道项上人头。金问道则是已知必败，一心想要逃。眼见金问道要败，北蛮部的战士同时冲了过来，要将金问道抢回本营。
<br/>　　野原火手下也沿着山坡冲杀过来，双方大军的交锋到此刻方才算正式开始。金问道落败之后，虽然拼死拼活被手下抢了回去，但己方却是士气衰落。与之相反，野原火的手下每个都是嗷嗷直叫，宛若下山猛虎，山林中的怪兽。
<br/>　　以野原火为首，他骑着一匹枣红马，领着大军宛若一柄长矛狠狠扎入敌方主阵，追着要杀金问道。
<br/>　　远远山坡上，柳褐岩眼底有些焦急，道：“大长老，再不下令，北蛮部怕就要败了。”
<br/>　　白无双奇怪的看他一眼，道：“为什么要下令？”
<br/>　　“这……”
<br/>　　“收兵，回城。”
<br/>　　白无双果断的下了命令，到此刻柳褐岩才大概明白，今天大长老只是过来看看，并没有要援助北蛮部的意思。
<br/>　　兵败如山倒，金问道一族之王，此刻还惶惶逃命，手下将士更加不会为了他去卖命。只见野原火带领的士兵气势如虹，轻而易举将对方主阵击溃，领着大军追杀金问道，反有人敢拦的，不过一刀斩了了事。
<br/>　　新王代替旧王，旧王不杀，新王如何登基？
<br/>　　金问道带着十余残骑，没了命的想前逃。汗水从额头渗出来，流进眼睛里，眼前模糊一片，竟已分不清自己逃到了那里。他只是一心往前逃去，溃败来得如此迅速，他的确没有想到。可冥冥当中，却又觉得这一幕早该会发生。
<br/>　　身后追杀自己的不像是野原火，而是那十余万的孤魂野鬼。原来自己终究没走出过那片山谷，日日夜夜里，那些尸体都会走入自己的噩梦中。梦，总有结束的时候，哪怕是噩梦。结束的方式，要么是从噩梦中挣脱，要么，是被噩梦拉入深渊。
<br/>　　金问道逃啊逃的，拼了命想要从噩梦中逃脱。
<br/>　　“金问道！”
<br/>　　身后一个声音突然响起，金问道下意识扭过头来，就见到一柄大刀迎面劈来。在刀刃上，看到自己的眼睛越来越大，咔嚓一声，大刀重重劈下。
<br/>　　一切都结束了。最后一刻，金问道忽然长长的松了口气，感觉一直缠绕自己的噩梦终于消失了。
