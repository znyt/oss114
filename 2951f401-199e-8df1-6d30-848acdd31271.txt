　　第712章
<br/>　　程大雷觉得『风寒』两个字太好用了，有什么事情自己不想做，随时可以把这两个字当借口。
<br/>　　借口有时候未必是为了蒙蔽对方，也同样可以表明自己的态度。不管李玉念怎么想这件事，只要他明白一点就行：送人质到长安城这种事想也不要想。
<br/>　　李玉念似乎也没报什么希望，道：“还是保重身体要紧。”
<br/>　　“是呐，长安居，大不易，她们不能住在长安，想也应该很遗憾。”
<br/>　　李玉念撇了撇嘴，似乎鄙夷程大雷的演技拙劣。略顿了顿，他开口道：“程先生想不想去一次长安？”
<br/>　　程大雷颦起眉头，困惑道：“此话怎讲？”
<br/>　　“我来凉州之前，陛下特意与我谈过。”李玉念叹口气，似乎想起离开长安城时的情景：“陛下提起，如果不是程先生镇守西北，如何能戳穿宋贼谋反的阴谋。如今，宋贼已送到长安城。”
<br/>　　程大雷一楞：“宋伯康还没死？”
<br/>　　李玉念眨眨眼睛：“自然没死，他落在百里侯爷手中，一直关押到现在，最近才押到长安。”
<br/>　　程大雷顿了一会，却原来宋伯康逃出凉州后，本意是要逃往长安。谁料半路被百里胜截住，任何人想来，他都是必死无疑。就连百里胜都是这么想的，他数次有念头要杀百里胜，为死去的弟兄复仇。
<br/>　　但杀了他又能如何，死去的兄弟也不能复生。百里胜不是不想宋伯康死，而是觉得即便杀死对方，也不足以泄恨。就一直扣押到现在，这次百里胜翻身成了功臣，宋伯康也被押到了长安。
<br/>　　李玉念道：“对于宋伯康这样的逆贼，自然万死不能赎罪，但对程先生这样的功臣，陛下也不能亏待。这次不仅是请程先生，也请天下诸侯到长安一会。一来是在天下人面前处决宋贼，二来也是昭告天下，封赏程先生。”
<br/>　　“什么事情？”
<br/>　　“八月十五，月圆之节。”李玉念道：“程先生到时候能去么？”
<br/>　　“呃……哈哈，这个，这个……”程大雷干笑两声：“不单单是贱内，在下身体也有些不舒服，至于到时候能去还是不能去，现在委实还说不好。”
<br/>　　双方又客套一番，李玉念最后说还要在凉州逗留几日，在城中逛一逛。看他的意思，估计还是刺探情报的目的居多。对此，程大雷也不以为意，反正君子坦荡荡，事无不可对人言，关键是程大雷也不觉得自己有什么秘密。
<br/>　　便是明摆着给你看，你又能怎样。
<br/>　　先安排李玉念等人下去休息，之后，程大雷与徐神机几人立刻凑在一起商量这件事。
<br/>　　“大当家，当真要去长安走一遭么？”刘悲道：“依我看这件事不妥，如果那李乐天动了杀心，大当家想从长安城平安脱身便难了。”
<br/>　　程大雷点点头，也明白刘悲话中的道理。可今次李玉念来到凉州城，先是要求程大雷派出人质，后又要求程大雷去长安，如果程大雷不走这一趟，『名正言顺』四个字就无法落实。
<br/>　　没有名份，总是一个尴尬的问题，也不能一直做个人人喊打的山贼。
<br/>　　“依诸位看，李乐天为何要请各路诸侯赴京？”程大雷问。
<br/>　　场上不仅有刘悲、徐神机、吴用，就连和珅也从朔方城请了过来。面对程大雷的问题，众人陷入沉思。
<br/>　　还是刘悲率先开口道：“怕是李乐天也要个名份。”
<br/>　　程大雷轻轻点头：“然也。”
<br/>　　程大雷占据凉州是有实无名，李乐天恰恰相反，他这个帝国天子是有名无实。帝国各路诸侯割据，图谋逐鹿，也恰恰是因为他这个有名无实。目前李乐天刚刚拔掉相府，算得上是有名有实。他也需要给各路诸侯看看，压下他们的不臣之心。
<br/>　　“那么天下诸侯会去么？”和珅问道。
<br/>　　程大雷道：“君令不奉，便着了一个反字，怕就没有在帝国立足的名份。”
<br/>　　“名份名份，名份两个字就那么重要么。”徐神机叹了一声。
<br/>　　“有时候就那么重要。”
<br/>　　刘悲此刻有些回过味来：“大当家是打算走这一遭了？”
<br/>　　“呃……哈哈，什么话，我这不正在与大家商量么。”
<br/>　　“大当家，长安凶险，若李乐天来一道绝户计，要将天下诸侯一网打尽？”
<br/>　　“怎么会，若是把各个山头的头目都赶尽杀绝，那帝国连表面的平衡也不能维持。那样一个帝国，并不是李乐天愿意看见的。”
<br/>　　“大当家该不会闲的无聊，想着去长安凑热闹吧？”吴用道。
<br/>　　“喔……哈哈，你可真会说笑。”
<br/>　　委实被吴用猜中，程大雷从始至终有一颗行走江湖的心。如果这次长安一会搞成，天下诸侯都会到长安，程大雷也想碰一碰世间英雄。有些诸侯名气很大，程大雷连面都没见过呢。
<br/>　　“大当家你可要知道，你这路诸侯，和其他人并不一样。”刘悲不满的盯着程大雷。
<br/>　　帝国二帝十八王，李星这位先不用提，程大雷这位凉州王，可是山贼出身，一直是帝国通缉的要犯。李乐天能放过其他诸侯，未必会放过程大雷。
<br/>　　当然，不去也有不去的坏处，李乐天毕竟是名义上的天子，不遵他的命令，就要抗旨的罪名，其他诸侯就可以名正言顺的攻打程大雷。
<br/>　　“依我看，大当家万万不可去。”徐神机忽然大手一挥：“我就提大当家走这一遭好了。”
<br/>　　程大雷吓得不轻，当真让徐神机赴长安，没有事也得被他搞出事来。
<br/>　　“徐军师息怒，息怒，如此大事怎么能够动用你呢。”刘悲道：“干脆我走这一遭好了，谅他姓李的也不敢对我怎样。”
<br/>　　派一个无关轻重的人去一趟长安，将程大雷凉州王的身份坐实，如此算是最稳妥的办法。
<br/>　　但除了程大雷之外，刘悲徐神机都没有什么本事，万一有个意外，二人绝对应对不来。
<br/>　　“罢了，罢了。”程大雷摆摆手：“先不要争，也不必吵，现在距离八月十五还有月余，我们慢慢商议不迟。”
<br/>(本章完)
