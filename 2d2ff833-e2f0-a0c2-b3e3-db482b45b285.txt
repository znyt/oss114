　　第64章
<br/>　　系统曾奖励一面大旗，附带招募技能，不过挂上去后，一直没有什么反应，程大雷都快把这件事忘了。
<br/>　　现在莫名其妙有两个人前来入伙，程大雷不得不想到这面大旗上。
<br/>　　他急匆匆离开大厅，来到校场查看大旗的信息，果然发现『招募』两个字变成了灰色。
<br/>　　原来是有CD的技能，可这CD时间也忒长了。
<br/>　　既然如此，两人的背景资料就不用查了，就像之前的秦蛮一样，忠诚度是没有问题的。
<br/>　　他又从外面跑回来，廖甲廖乙两兄弟也很奇怪，这位蛤蟆寨的大当家跑来跑去做什么。
<br/>　　“哈，二位能看上我蛤蟆寨，鄙人自然是荣幸之至，以后大家就是一家人……”
<br/>　　说着说着，程大雷脸上的笑容沉下脸，脸色不是很好看。
<br/>　　因为他看到了二人的信息。
<br/>　　姓名：廖甲（默默无闻的普通游侠）
<br/>　　年龄：29
<br/>　　技能：无
<br/>　　隐藏属性：无
<br/>　　姓名：廖乙（默默无闻的普通游侠）
<br/>　　年龄：27
<br/>　　技能：无
<br/>　　隐藏属性：无
<br/>　　合着是两个纯白板英雄，一无技能，二无属性，系统这招募技能可真够水的。程大雷忍不住想，会不会是因为『招募』技能的等级太低，以后就能招募到强大英雄。
<br/>　　当然，这只是程大雷的推测，究竟如何，还要靠后期一点点摩挲。
<br/>　　“黄总管，给二位兄弟登记，以后大家都是一家人。”
<br/>　　程大雷心情有些低落，不过聊胜于无，有了这二人的加入，山寨人口终于突破100大关。
<br/>　　嘟，『招募人口』任务达成，奖励抽奖次数1 ，是否现在抽奖？
<br/>　　NO。
<br/>　　程大雷拒绝了，人口达到100后，他立刻着手建设点将台。
<br/>　　他命停下山寨的训练，全体人手投入到这件事中。
<br/>　　说是点将台，其实就是一个土堆，上面夯土垒石，喜欢的还可以在上面立个神像，大家过来烧烧香许许愿什么的。
<br/>　　蛤蟆寨的吉祥物——LOGO是只蛤蟆，程大雷还没有恶趣味到在上面立一头蛤蟆，想着以后自己动员士兵，慷慨激昂，身后却有一只咧着大嘴的蛤蟆。
<br/>　　想想也是够尴尬的。
<br/>　　所以程大雷选择把校场的大旗立在上面，反正点将台的选址就在蛤蟆寨的最高处。
<br/>　　建设点将台最后只花了不到三天时间，完成后，程大雷把大家召集到这里，让徐神机发表了慷慨激昂的讲话。
<br/>　　程大雷终于发现了徐神机的可取之处，身为一个军师，徐神机简直就是个绝世毒奶，因果论武器一发动，无人能敌。不过，徐神机身上也是有一件长处，那就是——忽悠。
<br/>　　一说起话来，口若悬河滔滔不绝，有时候程大雷差点都信了。
<br/>　　既然如此，就要物尽其用，发挥他的特长。
<br/>　　嘟，点将台建设完成，山寨升级为二级山寨
<br/>　　嘟，山寨升级完成，山寨所有技能升级
<br/>　　嘟，山寨升级完成，拥有者等级提升1
<br/>　　嘟，山寨升级完成，奖励拥有者随机技能1
<br/>　　徐神机还在台上滔滔不绝的时候，程大雷收到一连串的系统提示。他赶紧回到自己屋中，检查系统信息。
<br/>　　山寨：蛤蟆寨（二级山寨）
<br/>　　拥有者：程大雷
<br/>　　人口：100
<br/>　　技能：齐心、治愈
<br/>　　原先的山寨一直是『破败的初级山寨』，当山寨人口扩充到100，以及建设完成点将台后，如今山寨终于升级成二级山寨。而且齐心、治愈、包括大旗附带的招募技能都获得升级，不过一些数值都是隐藏的，具体属性增幅多少，程大雷也不知道。
<br/>　　他又查看自己的信息。
<br/>　　姓名：程大雷（小有名气的优秀山贼）
<br/>　　年龄：18
<br/>　　技能：大魔王三斧、雄辩
<br/>　　隐藏属性：一个好人
<br/>　　恐惧值：8766
<br/>　　看到这里，程大雷有种如释重负的感觉，随着山寨升级，自己的等级也由普通提升为优秀，原先的三板斧变成大魔王三板斧。
<br/>　　就算现在遇到陆哼，也有把握一战，自然，输赢还不好说。
<br/>　　其实优秀已经很不错，百人里也未必能挑选出一个，像秦蛮那样顶级的实力，单打独斗可以横扫青牛山，放到军中率领一支千人众的士兵是没有问题的，不过秦蛮的智商……
<br/>　　好吧，这不是重点。
<br/>　　看来自己还是要把精力发展到山寨建设上，随着山寨建设，自己的等级也会获得提升。糟糕、普通、优秀、顶级、绝世、传说……秦蛮就是顶级山贼，绝世应该和大武流行的名将差不多，而传说大概就是张飞关羽那种等级的。
<br/>　　莫非自己以后也能匹马沙场，所向披靡，万军从中取上将首级如探囊取物……想到这里，程大雷隐隐有些激动，不过扛柄大斧子总觉得有些跌份。
<br/>　　吕布是方天画戟，赵云用的是龙胆亮银枪，宇文CD是凤翅镏金镋……古往今来，哪有名将是扛大斧子的，扛斧子最多就是李逵、程咬金这个级别，要么就是砍柴的。
<br/>　　找机会自己把它换了。
<br/>　　至于『雄辩』，应该是系统奖励的随机技能，程大雷把注意力搁在上面，却没有看到系统提示。
<br/>　　如此，就得自己琢磨了，这肯定是一项提升口才的技能。只是不知道，这雄辩提升的是辩论能力，还是忽悠能力。
<br/>　　这两者是不同的，前者就像诸葛丞相在阵前滔滔不绝，一句我从未见过如此厚颜无耻之人，然后王司徒就嗝屁了。
<br/>　　后者则是诸葛丞相滔滔不绝一番，然后王司徒在阵前归顺了。
<br/>　　之后，程大雷又开始检查点将台的信息。
<br/>　　在点将台建设完成后，系统出现一座缩小的点将台模型，当程大雷将注意力集中在上面是，脑海中出现一行系统介绍。
<br/>　　初级点将台：每10000恐惧值抽取一次，随机获得人物1 。
<br/>　　嘟，你的威名口口相传，在大陆上传播，四方各地的英雄将慕名而来。
<br/>　　这几天程大雷获得的恐惧值一直在增长，这应该是自己做的坏事传出去后散开的恶名，目前已积累到八千多，距离抽奖还差一些，不过过几天应该就能凑够。
<br/>　　不过，自己还有一次系统奖励的抽奖机会，程大雷好久没有抽奖了，不知道自己有没有脱非入欧。
<br/>　　“抽奖！”
<br/>　　程大雷洗手净面，恭恭敬敬点了三炷香。
<br/>　　
<br/>(本章完)
