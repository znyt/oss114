　　第658章
<br/>　　在苏樱处饮茶，在樊梨花这里便只能饮酒。二人摆开杯盘，闲饮叙事，樊梨花最近过得无趣，平素在院子里舞枪弄棒也不太像话。
<br/>　　“喂，不如你分我一个差事做？”樊梨花道。
<br/>　　程大雷刚端起一杯酒，这话确实也不是樊梨花第一次说，他道：“不是我不想应你，委实是没有你能做的事。上阵打仗，我也怕你有了闪失不是，可其他事你也未必做得成。”
<br/>　　程大雷倒也不是应付樊梨花，自己手上不缺人手，关鱼和秦蛮都是绝世武将，有万夫不敌之勇，赵子龙最近也快突破了。樊梨花曾经很强，现在她的实力在蛤蟆城却不算什么。
<br/>　　“我不管，我总是要做事的。”樊梨花托着香腮，口中吐出一道幽兰：“你不派我差事，我就自己找，实在不行，我便回我的落玉寨。”
<br/>　　落玉寨由高力士和徐灵儿负责，算是蛤蟆城的马场，程大雷道：“到落玉寨你又能做什么事，到时候更难见到我了。”
<br/>　　“在这里平素也见不到呐。”樊梨花有了醉意，也有了愁绪：“你一去三五月，当真是死在外面我们也不知道。”
<br/>　　说罢便觉得失言，忙啐了两口：“呸呸，打嘴。”
<br/>　　程大雷心头一暖，心底也有些愧疚。他这般人，在刀尖上打滚，的确是某天走窄了，死在别人明枪暗箭之下也算不得什么稀罕事。自己与樊梨花和苏樱的相处其实并不算愉悦，但二人心里终归是惦记自己的。
<br/>　　最后他将杯中酒饮干，道：“好了，你愿意做什么便做什么，不惹什么岔子就好。”
<br/>　　“废话我能惹什么岔子。”
<br/>　　二人一边饮酒一边谈话，待樊梨花快醉时，他将樊梨花抱在床上，为她盖上薄被。之后便去了苏樱的院子，待到后半夜，便又从苏樱处出来，再重新回到樊梨花处。
<br/>　　一来二去，其实十分麻烦。但既然身在这个位置，有些麻烦事他也是必须去做的。
<br/>　　只等樊梨花睡熟了，他才悄悄披衣起身，来到前庭自己的书房。窝在炕上，就着一盏油灯，翻看蛤蟆城最近发生的事。
<br/>　　程大雷并非工作狂，但只有在此地，他方才能得几分清净，算是属于他的时间。曾经总想着左拥右抱，尽享齐人之福，但程大雷到此刻才明白过来，齐人之福也没那么好享。
<br/>　　无论是樊梨花还是苏樱，总给他一种难以描述的压力，不能说是坏事，但总归是压力。
<br/>　　其实最近蛤蟆城也没什么大事，大家也就是种田，练兵，再种田再练兵。谁偷了谁家东西，谁调戏了谁家姑娘……只要人一多，便必定有这种事发生，程大雷治下也并非人间天堂，蛤蟆城也不是世外桃源，无非就是普普通通一座城，有好有坏的人凑在一起过日子而已。
<br/>　　这些事杨国忠处理得都不错，程大雷也就是借此打发一下时间，躲躲清净。
<br/>　　“大当家，我看书房亮着灯，便与你过来送壶茶。”
<br/>　　清净被打破了，柳芷端着茶进来，曾经这位柳家的大小姐，如今是当真诚心给程大雷为奴为仆。
<br/>　　程大雷倒是没太好意思赶柳芷出去，虚伪的应承了两句，便继续翻面前的书卷。
<br/>　　除了苏樱的事让他有些头疼之外，程大雷也必须留出一些时间，想一想蛤蟆城的下一步该如何走。
<br/>　　帝国乱世，诸侯割据，这是大局势，免不得日后就是群雄逐鹿，争霸天下。而自己，究竟该在这场大局中扮演怎样的角色呢。
<br/>　　程大雷一开始也没敢太贪图，只求经营好蛤蟆城，有在乱世中自保的能力便可。但华氏一族的事情提醒了他，自己从前的想法或许太过天真。
<br/>　　这场乱世赌局中，你要么不上牌桌，既然上了牌桌，就要有准备拿走全部，否则，就是输掉全部，至于明哲保身，则是幼稚的妇人之见。
<br/>　　可程大雷并不是个有野心的人，当年在青牛山他委实是想做些事，后来经历的事越来多，知道很难，于是也就消了这心思。
<br/>　　可现在这个局面，如果程大雷说自己没有野心，不在牌桌之上，怕天下诸侯一个信的没有。
<br/>　　也是胡思乱想，目前还只能走卷，偶尔柳芷上前将茶杯倒满，或者是将冷掉的水煮开，红袖添香确实也算惬意。
<br/>　　到五更天的时候，程大雷已经有些困意，他伸了个懒腰，无意中扫了一眼柳芷。
<br/>　　“那个……你是不是有什么话想说？”
<br/>　　程大雷也不傻，这姑娘立在一旁半天，就没有一刻消停过，当是心里有事。
<br/>　　“确实有件事想说，只是不知该不该讲？”
<br/>　　“那就不要说了。”
<br/>　　“呃……”
<br/>　　柳芷和苏樱曾经都是千金小姐，有同一个毛病，喜欢假客气。程大雷无语的摆摆手：“好了不逗你了，想说什么便说吧？”
<br/>　　“那个……想请大当家准柳芷一门亲事。”
<br/>　　程大雷本来很自然，委实也算是风轻云淡，大家风范，但听到这话心里忽然有些不是滋味。
<br/>　　“那个……和谁呀？”
<br/>　　“也不是太要紧的人，是军里一个小卒，大当家怕连他名字都不知道。”
<br/>　　程大雷轻轻点头，蛤蟆城人是多的，他连城主府的下人都认不全，何况是军中的普通兵丁。
<br/>　　“这个是不是有些委屈你，你若想嫁，军中还是有些青年才俊的，一营之长，一军之将，他们还不是任你挑选。”
<br/>　　的确，程大雷手中也算人才济济，尤其是关鱼训出来的年轻小将，个个严肃庄重，都是俊品人物。
<br/>　　程大雷说这些话的时候，只想打自己嘴，但这些话他也不得不说。他现在只想劝柳芷缓一缓，等一等，心里想的却是：究竟是那个王八蛋。
<br/>　　柳芷忽然苦涩的笑了笑，道：“一营之长，一军之将，奴不过是残花败柳，那军中的才俊如何能看得起我。”
<br/>　　“这话可是从何讲起呐。”
<br/>　　程大雷长叹一声，目光看着柳芷的表情。忽然觉得事情有些不对，或许说自己一直忽略了很多事。
<br/>(本章完)
