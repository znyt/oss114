　　第626章
<br/>　　乱世牌局，无论程大雷想或不想，事实上，他已经在这张牌桌上落座。
<br/>　　帝国之大，怕是哪一家势力都不会忽略蛤蟆城。如果有可能的话，宋伯康第一个想除掉的人并不是百里胜，绝对是程大雷。无非是忌惮蛤蟆城精兵强将，他暂时没有动手罢了。
<br/>　　能在这张牌桌上占有一席之地的，中间不会有傻瓜。其实很多人都抱着和程大雷同样的心思，悄悄扩充势力，闷声发大财。
<br/>　　可在扩充势力的过程中，难免就会侵占其他人的地盘，最后谁也不肯相让，那也就只能靠战斗来解决了。
<br/>　　宋伯康是名义上的凉州王，可在他控制的地盘，不仅有程大雷这心腹大患，还有百里胜这样的存在。
<br/>　　眼看这两家的势力越来越大，如果再不动手，这凉州究竟是谁的地盘，也就不好说了。
<br/>　　百里胜是将军府扶持的势力，从一开始宋伯康就想除掉他。曾经还想联络程大雷，两家合力吞掉百里胜，只不过程大雷没被他忽悠住罢了。
<br/>　　这一日，宋伯康在自己王府中思考如何用兵。陆洛快步走进房间，道：“王上，刚刚得到的消息，程大雷已经返回蛤蟆城。”
<br/>　　宋伯康一惊：“当真？”
<br/>　　“千真万确，我们的探子亲眼看到他入城。”
<br/>　　宋伯康皱起眉头，心中在思量着这件事。他想打掉百里胜，其实这件事并无什么困难。百里胜镇守边关，只能依靠帝国的粮草支援，不像蛤蟆城一样已经可以完成自给自足。只要自己切断他的粮道，用不了多长时间，他便会不攻自破。
<br/>　　关键还是程大雷的态度，如果程大雷从中作梗，这件事会增加很多变数。以他的人性究竟是会按兵不动，还是会暗中相助百里胜，这件事宋伯康一点都看不出来。
<br/>　　如果他真打算使坏，等自己上房抽梯子都算是好的，假若趁着城中兵马空虚，一股脑打过来，也不是一件不可能的事情。
<br/>　　宋伯康一直犹豫不决，没有对百里胜出兵，忧虑的就是这件事。
<br/>　　“王上，属下有一计，不知当讲不当讲。”陆洛道。
<br/>　　“喔，你有什么计策啊？”宋伯康不是很看得起陆洛，如果不是仗着家世，自己本不屑用他。
<br/>　　陆洛深吸一口气，自从上次贸然出兵被程大雷击溃后，他的地位就一落千丈。现在他急需做件事，挽救自己的政治生命，证明自己的价值。
<br/>　　“不如我们让给程大雷一座城池。”
<br/>　　“什么。”宋伯康一惊：“城池怎么可以轻易让给别人，我看还不如把整座凉州让给他，我自己去给程大雷牵马好了。”
<br/>　　“王上勿怒，听我把话说完。”这件事陆洛已经考虑良久，此刻娓娓道来：“蛤蟆城被程大雷经营多年，如今城中粮米如山，强将如云，俨然铁桶相仿。以我们目前的实力，先不说能不能打下蛤蟆城，就算打得下来，也必定损兵折将严重，得不偿失。”
<br/>　　“谁说不是呢，如果可以打下蛤蟆城，我会向百里胜动手么。”宋伯康长叹一声，道：“你继续说？”
<br/>　　“反正也打不下蛤蟆城，索性就将他引出来。给他一座城池为饵，程大雷绝对不会拒绝。可要经营一座城池，他必定要派出人手，而城中的平民也会大大消耗其的粮草。”陆洛慢悠悠道：“两只拳头都想握住东西，有时候就是什么都得不到。”
<br/>　　刚开始听着陆洛的话，宋伯康不以为意，但渐渐他眯起了眼睛。陆洛口中所说，未必不是一个办法呐。既然程大雷强大，就索性分散他的势力，一来可以让他保证在这次行动中按兵不动，二来分散他的力量后，也方便日后自己消灭他。
<br/>　　“我觉得程大雷也不会拒绝，关键是……他能够信守承诺，得到城池后按兵不动么？”这是宋伯康考虑的一点，乱世天下，信用是一件很可笑的事情，宋伯康不相信程大雷会拥有这种东西。
<br/>　　陆洛双手一抱拳：“臣下愿意亲自走一趟蛤蟆城，动用三寸不烂，说服程大雷。”
<br/>　　帝国很大，程大雷的名声很糟糕，天下人传说他是如何十恶不赦，每顿饭要吃十斤人肉，有一百个老婆云云。但凉州城距离蛤蟆城颇近，双方探子不停打着交道，所以凉州方面对程大雷还是有些了解的。
<br/>　　他并非传说中那样残暴，本人也并不嗜杀，目前一妻一妾，生活方面也算是检点。总的说来，程大雷还是个正常人。对于来到蛤蟆城，大家已没有之前深入魔窟的感觉。
<br/>　　为了表示诚意，陆洛单人独骑来到蛤蟆城。说明来意后，被引到城主府，见到了程大雷，一旁坐着刘悲、和珅、徐神机几人。
<br/>　　一番客套之后，陆洛说明了自己的来意。为了程大雷按兵不动，凉州方面愿意将朔方城让给程大雷。
<br/>　　程大雷一直在揣测陆洛此行的目的，所谓无事不登三宝殿，他来绝不会只是向自己问号。然而，听罢陆洛的话后，程大雷还是大吃一惊。
<br/>　　宋伯康究竟是什么心思，会送给自己一座城。
<br/>　　“那个……我家王上还有一个要求？”
<br/>　　程大雷心中明白，宋伯康也不会平白给自己一座城。他微微倾了倾身子：“凉州王需要我做什么？”
<br/>　　陆洛道：“咱们两家共治凉州，还需程当家派出一人从中联络两家的合作事宜。”
<br/>　　程大雷眨了眨眼睛，理解了陆洛的话。为了保证自己不会出兵，自己需要给陆洛一名人质。
<br/>　　这也在情理之中，不然对方不会白白的送给自己一座城，自己拿着也不放心。
<br/>　　可派谁去做人质，倒还是个问题。其实也算不上什么问题，程大雷的目光自然而然落在徐神机身上。
<br/>　　“徐军师，不如你走一趟？”
<br/>　　徐神机一激灵，他再糊涂也明白做人质是什么结果。慌着直摆手：“大当家，老朽一把年纪，受不了这颠沛之苦。这样艰巨的任务，你还是交给别人吧。”
<br/>(本章完)
