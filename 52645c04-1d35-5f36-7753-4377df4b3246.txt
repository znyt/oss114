　　第678章
<br/>　　根据情报，程大雷已知道崔白玉身边有三名贴身的丫鬟，不仅容貌无双，而且每个人都有出奇的本事。但没有任何一份情报告诉程大雷，这三人都有特别的武艺。
<br/>　　当真藏得够深。
<br/>　　刚才三人默契配合，打了程大雷一个措手不及。也是程大雷反应不慢，才没有丢掉性命，否则，这一刀砍得便不是他的胳膊，而是他的喉咙。
<br/>　　三人各自握着一柄短刀，拦在程大雷面前，虎视眈眈如小兽一般模样。
<br/>　　程大雷手摁住伤口，舔了舔指尖的血，忽然哈哈大笑。
<br/>　　“萤烛之光，也敢与日月争辉，来来来，让本当家好生调教调教。”
<br/>　　三人一个眼神传递，便同时跃起，向程大雷攻来。手中短刀闪过寒芒，一寸短一寸险，当他们迫近时，凶险的便是程大雷。首先程大雷的斧头便施展不开。他右手握着斧头，只能用左手战斗，实力被大大限制。
<br/>　　这三人窜上跃下，如穿花蝴蝶相仿，攻击一环扣着一环，没有半分松懈。
<br/>　　便是观战的高飞虎、高飞豹、小白狼三人都倒抽一口凉气。大当家一着不慎，莫在阴沟里走窄了。
<br/>　　程大雷不得不承认，的确小看了三人。但这三名小丫鬟想要伤到他，的确有些异想天开。稍稍镇定下来，程大雷左手探出，如恶龙出海相仿，一把握住其中一人的手腕。
<br/>　　此人是小鹿，一个瓜子脸的美女，谁料她一翻身，身体竟以难以相信的弧度折叠，双脚踹向程大雷的胸口。
<br/>　　同时，刀与犬的攻击也到了。
<br/>　　程大雷倒抽一口凉气，不得不松手，在间不容息的关头，攥住了小鹿的脚踝。
<br/>　　根本没给她反应的机会，直接抡开，竟然用身体做武器，挡住了刀与犬的攻击。
<br/>　　人力有时而穷，这名叫小鹿的丫鬟虽然强悍，但在空中抡了一圈，也是七荤八素。
<br/>　　程大雷将她往后面一丢，口中道：“绑了，押回山寨去。”
<br/>　　小犬与小刀不得不退下，目露警惕，此贼名不虚传，果然有了得之处。
<br/>　　程大雷此刻已经将大斧放在背上，好整以暇，伸开双臂，冲两女招招手。
<br/>　　“来，来，来。”
<br/>　　两女同时跃起，向程大雷发动攻击。若是跃起扑火的飞蛾。
<br/>　　程大雷伸展双手，几乎同时抓住二人双手，接着在中间狠狠一撞，二人额头撞在一起，顿时头破血流。
<br/>　　程大雷将她们往后面一丢，道：“绑了。”
<br/>　　接着，程大雷目光落在崔白玉身上，桀桀冷笑道：“传说崔姑娘貌美无双，凡见过的人都生不如死，今天本当家倒是要冒冒险。”
<br/>　　崔白玉冷目望着程大雷，忽地从袖子中伸出手，寒芒一闪，狠狠扎向自己胸口。
<br/>　　那柄带在身上的匕首终于发挥作用，因为她明白，一旦落在程大雷手中，要比死了更加恐怖。
<br/>　　锵！
<br/>　　匹夫剑在这一刻出鞘，电光火石间将崔白玉手中的匕首击落。崔白玉愣了愣神，手中空空荡荡，只是虎口有些酸疼。
<br/>　　程大雷吹了声口哨，笑道：“本当家的匹夫剑出鞘诸多次，杀人无算，出鞘为救人，你是第一次。姑娘，你有福了。”
<br/>　　说着，程大雷抬起剑，将她面上黑纱聊下。一面黑纱轻飘飘落地，所有人的视线都落在崔白玉脸上，接着便是齐刷刷的吸气声。
<br/>　　也不单单是程大雷好奇，诸多人也都想知道这白玉美人究竟生得如何模样。便是崔白玉身边的亲卫，也没有一人见过她的模样。
<br/>　　程大雷下意识皱起眉头，崔白玉以黑纱遮面时，眉目清冷，眼里像是藏着冰。可黑纱落下，她两腮微微的婴儿肥，何止不清冷，简直有几分可爱。此刻，冲程大雷怒目相视，不用怀疑她想杀程大雷的心。但这副表情，怎么看，怎么感觉像个糖被别人偷走的孩子。
<br/>　　她一直用黑纱遮面，莫非就是为隐藏自己的可爱气质。的确，这样一张婴儿肥的面庞，当真没有什么气场。
<br/>　　程大雷口中啧啧出声：“崔相上辈子积了什么德，竟然生下你这等姑娘，当真是老天爷不开眼呐。”
<br/>　　崔白玉终于忍不住，口中吐出两个字：“狗贼！”
<br/>　　程大雷哈哈大笑，忽然伸出手，将崔白玉捞在自己怀中，口中怪叫一声：“你便随了本当家，去做我的压寨夫人，先给我这狗贼生几个小狗贼。”
<br/>　　程大雷顺势在崔白玉怀里鼓捣了一把，感觉那沉甸甸的重量。心中感慨：这姑娘原来这么大的本钱，只是一身宽大袍子，倒是看不出来。
<br/>　　崔白玉又羞又怒，可她被揽在程大雷怀里，身体动弹不得。一股裹夹着血腥的汗臭味扑面，熏得她头晕眼花。
<br/>　　这种憋闷至死的感觉并没有持续多久，程大雷将她丢在地上，口中道：“绑了，她若有个闪失，我拿你们是问。”
<br/>　　立刻有两个小喽啰过来，将崔白玉紧紧绑住。面对她的倾城之貌，的确心慌意乱，有种亵渎神明的感觉。
<br/>　　程大雷也不可能在崔白玉身上浪费太多时间，他更在意的是宋伯康。宋伯康不死，就不能说此战获得了胜利。
<br/>　　程大雷收束起城中的兵丁，向守军驻地攻去。凉州城的守军都驻扎在东门，这便是程大雷故意放开东门的原因。
<br/>　　只盼着自己在城中搞出如此大的声势，那些守军为望风而逃，真如果他们负隅顽抗，程大雷当真不好对付。
<br/>　　毕竟，杀入凉州城的其实只有三千人。
<br/>　　程大雷领着兵丁冲到东门，与赵子龙、史文恭、秦蛮等人汇合。
<br/>　　和程大雷期待的不同，守军并未撤退，仍然窝在兵营中。如今，已经被蛤蟆城的人马包围。
<br/>　　当然，以三千人对十倍己方的兵力，也勉强算得上包围。
<br/>　　“大当家，宋伯康逃了。”赵子龙道。
<br/>　　“咦！”程大雷一楞：“真的逃了。”
<br/>　　“属下亲眼所见，他的确带着亲卫从南门逃走。”
<br/>　　程大雷毕竟高估了宋伯康，这是一头老狐狸。但狐狸这种生物从来不缺乏智慧，只是缺乏勇气。
<br/>(本章完)
