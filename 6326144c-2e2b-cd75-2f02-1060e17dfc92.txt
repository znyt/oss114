<br/>    不过半天的时间，陈明已经来到了一座莲花台之上，上面有着十多个小侍女，朝着陈明恭敬行礼道：“上师请！”
<br/>    
<br/>        陈明在莲花台上的一片荷叶上坐下，金镶玉拍拍手，然后道：“大祭司既然是修心僧，为我瑶池尽心尽力，我瑶池自然是不可能亏待上师，来人，将蟠桃送来供上师享用。”
<br/>    
<br/>        不过片刻，有一个老人托着一个玉盘登上莲花台，陈明隐隐可以闻到一股摄人心魄的桃香，脸上满是皱纹的老人来到陈明的身前，朝着陈明弯腰行了一礼，而后道：“燃罪上师请看，这便是我瑶池的蟠桃，这一颗最大的，乃是瑶池的万年蟠桃，吃下这一颗蟠桃，足足可以增长一万年寿元！”
<br/>    
<br/>        陈明微微点头，老人继续介绍道：“而旁边的这一颗，便是瑶池的千年蟠桃，吃下之后，可以增长一千年的寿元！至于其他两颗，便是百年蟠桃和十年蟠桃。”
<br/>    
<br/>        “请上师享用。”
<br/>    
<br/>        陈明心说这瑶池就是懂事啊，上来就是这般重礼！
<br/>    
<br/>        陈明面上不动声色，拿过一颗蟠桃，放在嘴里，咬了一口，桃香四溢，满口都是仙桃的味道，清脆可口，很是爽口。
<br/>    
<br/>        不过一盏茶的功夫，陈明已经吃下了一颗万年蟠桃，陈明能够感觉到，自己浑身都充满了活力，这蟠桃，果然是有着逆天改命的能力，陈明感觉浑身都是一阵轻松，如同飘了起来一般。
<br/>    
<br/>        冥冥之中，陈明能够感觉到，自己的寿元似乎变得绵长无比。
<br/>    
<br/>        之前的陈明，也就是一个悟道境，三百年的寿元了解一下，而现在，陈明的寿元，是万寿无疆！
<br/>    
<br/>        将剩下的三颗蟠桃吃下，金镶玉很是期待地看向陈明，问道：“上师，不知道这瑶池的蟠桃如何？”
<br/>    
<br/>        陈明微微一笑，然后道：“味道不错，就是没吃饱，要万年的可能有点过分，把那个十年的再送几颗我吃个饱。”
<br/>    
<br/>        金镶玉：........
<br/>    
<br/>        老人：........
<br/>    
<br/>        谁吃蟠桃看好不好吃啊，你还没吃饱！
<br/>    
<br/>        金镶玉尴尬地笑笑，“来人，再去取十颗十年的蟠桃过来。”
<br/>    
<br/>        年的蟠桃而已，陈明如今的身份，再吃一些，虽然无用，但是也不过分。
<br/>    
<br/>        待得陈明吃了个饱，陈明这才道：“带我去看无字碑吧，我身兼四国大祭司，无法在瑶池久留。”
<br/>    
<br/>        金镶玉这才叫来伴月婆婆，送陈明去看看无字碑。
<br/>    
<br/>        待得陈明离开，金镶玉这才看向身旁的老人，问道：“九千岁，你看此人，如何？”
<br/>    
<br/>        这个老人，竟然是九千岁，刚刚却是在扮做一个小厮，伺候陈明！世人皆道九千岁已经被瑶池囚禁，如今看来，倒是虚言。
<br/>    
<br/>        九千岁呵呵一笑，然后道：“此人，宵小也！”
<br/>    
<br/>        金镶玉道：“那我们刚刚还给他吃了万年的蟠桃！岂不是喂了狗吃了？”
<br/>    
<br/>        九千岁大笑道：“正是这宵小之身，荣登四国大祭司，才显得他已经到了更高的境界，他如今，玩弄的，是人心。此子，堪称，四个字：风华绝代，我倒是有些期待他打开局面之后，和他对弈一局了。”
<br/>    
<br/>        金镶玉道：“在九千岁看来，他到底想要什么？”
<br/>    
<br/>        九千岁微微摇头，“看不穿，这才有意思，已经知道胜负的棋局，便没有任何意思了，放长线，才能钓大鱼，且放线让他随意去折腾吧。”
<br/>    
<br/>        陈明站在一座莲花台之上，其中，一朵朵莲花的花瓣打开，露出三十二块无字碑，陈明看向这些无字碑，双目之中闪烁着神光，他还差三十二颗道种，而这里，便是有着三十二块无字碑。
<br/>    
<br/>        陈明的体内，最后三十二座道宫凝聚而出，陈明感受着体内的三千座道宫沉浮，即便是四大仙国把持着这世间的一切，陈明，终究还是修成了这无量道体了！
<br/>    
<br/>        占天养此刻正在吃着萝卜和腊肉，腊肉最好是和萝卜一起吃，这样才不会腻。
<br/>    
<br/>        就在这时，天空之中忽然传来了一声巨响，占天养冥冥之中感受到了一阵惊变，他走出房间，看向天空之中，一片天际破碎了开来，那是一座巍峨的神山一角出现在眼中。
<br/>    
<br/>        占天养手中端着的碗落下，在他的脚下摔得粉碎，他看向天空中那巍峨的神山一角，那是什么？
<br/>    
<br/>        周围的星空都被压得如同快要破碎，索性只
<br/>    
<br/>        是神山的一角，而不是所有。
<br/>    
<br/>        占天养忽然想起一个他师祖的师祖曾经说过的传说，天地之间，有大修为者，若是修成了旷世绝伦的神功，便会有无字劫落下。
<br/>    
<br/>        而这无字劫的其中一环，便是传说中的人间劫！
<br/>    
<br/>        而这，同样被称作无字咒，乃是这天地之间最为可怕的诅咒。
<br/>    
<br/>        想要修成这世上最为可怕的神功，就必须经受整个世间的考验！
<br/>    
<br/>        这种修行者，走的乃是逆天的修行之路，故而天地会降下劫数，这其中，便是有着最为可怕的人间劫。
<br/>    
<br/>        所谓人间劫，便是其他的修行者可以在天劫之下进攻渡劫者，而不会被天劫波及。而这神山的一角，将会出现在每一颗星辰之上，让所有满足条件的修行者都能进入其中。
<br/>    
<br/>        说白了，就是天劫就是要用尽一切在规则内的手段灭杀了这个渡劫者。
<br/>    
<br/>        想要掌握这世间最可怕的伟力，窃取天道的权柄，便要经受天道的考验。
<br/>    
<br/>        占天养忽然想起了一句话：人欲称王无字始，一袭锦袍夜不归！
<br/>    
<br/>        “这世间，又要有新的变动了。”
<br/>    
<br/>        天庭。
<br/>    
<br/>        本是闭目养神的帝主睁开双眼，看向天空之上的那一角神山，叹道：“多事之秋啊，没有想到，现如今，居然还有人在觊觎天道的权柄。”
<br/>    
<br/>        “不过，这一次，倒是可以看看各家的底牌，这是一次豪赌，对方已经下了赌局，赌的是整个世界，我想，没有一家会藏着自己的底牌吧。”
<br/>    
<br/>        太虚。
<br/>    
<br/>        虚空忽然裂开，一朵虚空花骤然在神山的一角开放，而后瞬间消失在虚空之中，只留下一句淡淡的话语，“传命，让虚空太子出关。”
<br/>    
<br/>        瑶池。
<br/>    
<br/>        九千岁忽然在此刻抬头，看向那神山的一角，“无字劫来了，谁的无字劫？”
<br/>    
<br/>        九千岁掐指一算，然后摇头，“算不出来，算不出来的事，才能显得出本事。”
<br/>    
<br/>        九千岁能够感受到背后有着人来，那是金镶玉，九千岁没有回头，只是道：“准备吧，来者不善！”
