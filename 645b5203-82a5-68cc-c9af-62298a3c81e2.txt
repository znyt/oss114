　　第127章
<br/>　　过去这段时间，程大雷的恐惧值一直在涨。就像一颗石头落入湖面，惊起的波澜要许久方得平静。比如有小孩啼哭不止，母亲便吓唬他：若你再哭，便让蛤蟆大王捉回去吃了。
<br/>　　如此，程大雷就又能收割点恐惧值。
<br/>　　一日复一日，恐惧值每天都在涨，今日突破10万。
<br/>　　每10万可连抽十一次，必得优秀品质1。每100万可抽取顶级品质1。上次赵子龙就是这么抽出来的，在之后的行动中发挥了很大作用。所谓千军易得，一将难求，一位顶级英雄是比一百喽啰有用。
<br/>　　只不过，拥有10万恐惧值而不体验一下十一连抽，总感觉生命好像不完整似的。
<br/>　　程大雷决定抽奖，哪怕只体验一次，以后再得恐惧值自己就好好攒着，兑换顶级英雄。
<br/>　　山寨的点将台已搬到山谷中，代表蛤蟆寨的大旗已经竖起，就是那副上面画着蛤蟆顶个蛋用，却被徐神机唤做天蟾吞月图的大旗。在过去这段时间，大旗的招募技能又生效过一次，帮山寨招募到一人。只不过对方没有什么出奇之处，于是程大雷也没在意。
<br/>　　“点将。”
<br/>　　程大雷心中默念这两个字，脑海中点将台的图像清晰起来，篝火燃起，火光明灭间，显出一个个人来。
<br/>　　嘟，获得糟糕游侠1……
<br/>　　嘟，获得普通游侠1……
<br/>　　嘟，获得糟糕猎人1……
<br/>　　嘟，获得普通农夫1……
<br/>　　……
<br/>　　……
<br/>　　嘟，获得优秀铁匠……
<br/>　　程大雷捂脸，看来自己依旧是非人的命，脸黑得令人发指。十一连抽，结果连农夫这种职业都搞出来了。
<br/>　　在一堆白板英雄中，总算见着个挂色的，出现一名优秀铁匠。当然，这些人目前正在来的路上，说不好什么时候会到。程大雷先通过系统查看这铁匠的信息。
<br/>　　猪铁匠：精通锻造的优秀铁匠，马家村牛铁匠的传人。
<br/>　　拥有这样一个姓氏，而且连名字都没有，想也不是什么重要人物。不过，『马家村牛铁匠』这六个字，却令程大雷感觉有些熟悉，好像在那里见过。
<br/>　　他拿出自己的斧头，这斧头升级后变成了鬼面斧，成为绝世等级，附带技能：热血。
<br/>　　果然，鬼面斧的信息中有一条：马家村牛铁匠所制。
<br/>　　本来程大雷没留意这条信息，还以为是系统随手加上去的，无足轻重。但如今看来，这马家村的牛铁匠能锻造出绝世等级的兵器，教得出优秀等级的徒弟，很明显来头不小么。
<br/>　　难得是一位隐藏npc 。
<br/>　　游戏中经常可以遇到这样的人，庙后的扫地僧人，缺门牙的老黄……看上去平平无奇，一开口却是：俺当年屠龙的时候……
<br/>　　不知日后有没有机会见到。
<br/>　　程大雷同黄三元说了一声，若有一个姓猪的铁匠来投奔，就让对方来见他。他确实要打听打听这位马家村的牛铁匠。
<br/>　　……
<br/>　　弹指一挥间，便又是月余光景。天气是一日赛一日的冷，话说这北方的冬天当真是漫长。到后来，冷得实在没办法干活，程大雷只好让大家停下来，一切等明年开春继续。
<br/>　　程大雷收到苏樱从落叶城捎来的两封信，信中说苏樱父亲的病一日更甚一日，怕是撑不过这个冬天。苏樱本想亲自去看一看，最后却连苏家的门都没能进去。在信中苏樱提起此事，也是难掩伤心。
<br/>　　冬天是鬼门关收人的日子，多少老人熬不过这个冬天，这种事程大雷也没有办法。
<br/>　　及到这一天，青牛山突然迎来一伙客人。
<br/>　　蛤蛤蟆寨已经搬进山谷，诸多山寨有些已经拆了。这伙人去的是飞虎寨，或者说，他们本就是奔着高飞虎来的。
<br/>　　程大雷去的时候，见飞虎寨聚义厅内，高飞虎、小白狼、熊大熊二等各山各寨的首领都在，他们坐在大厅左侧一排椅子上。而右侧则坐着四位裹着大袍，身材高大的生面孔。
<br/>　　在他们双方中间，那张代表当家话事之人的太师椅空着。
<br/>　　程大雷信步走到那张椅子前撩袍坐下。
<br/>　　高飞虎几人同时站起，拱手齐声道：“大当家！”
<br/>　　连带着四位客人也站起，心里琢磨着：青牛山主事的不一直是高飞虎兄弟么，什么时候换了这年轻人。
<br/>　　“坐，坐。”程大雷没有起身，随意摆了摆手。
<br/>　　他望向那四位客人，道：“诸位我之前没有见过，还未请教几位贵姓？”
<br/>　　“某乃幽州齐德强，这次奉主公之命，特来拜会青牛山的诸位英雄。”
<br/>　　“阁下的主公是？”
<br/>　　“幽州杨龙停。”
<br/>　　程大雷目光移开，在高飞虎几人身上转了圈，他们此时都看着程大雷，看来已经知道齐德强的来历。
<br/>　　“齐将是不是有个弟弟叫齐德龙啊？”
<br/>　　齐德强一愣：“某正有个哥哥名叫齐德龙，同样在主公座下任职，阁下是怎么知道的，难道阁下认识我兄长？”
<br/>　　“官匪不同炉，我怎有幸认识令兄。”程大雷笑笑，奇得隆咚镪嘛。他继续问齐德强，道：“幽州王的威名吾辈也有耳闻，只是我等绿林草莽自然入不得幽州王的眼界。两家也素无恩怨纠葛，这次……幽州王怎么想起我们这些人来？”
<br/>　　“阁下谦虚了，绿林之中多豪杰，主公提起青牛山的诸位英雄，也是赞不绝口。这次主公亲笔修书一封，让我转呈青牛山当家的。”
<br/>　　齐德强从怀里掏出一封信，看看程大雷再看看高飞虎。
<br/>　　高飞虎立刻道：“这位是程寨主，青牛山诸事都是程寨主当家。”
<br/>　　“便是那人送外号蛤蟆大王的程当家么，大名久仰，却是没想到竟是如此年轻，当真是英雄出少年。”齐德强把信递到程大雷手中。
<br/>　　程大雷拆开信，齐德强返回座位，冲众人道：“戎族屡屡犯我边关，烧杀劫掠，无恶不作，近些年却是愈演愈烈。诸位虽是草莽，却同样是我帝国子民，既然有一身本事，好男儿自然该投身军务，报效国家。”
<br/>　　程大雷缓缓把信合上，抬起头问：“幽州王的意思是……招安？”
<br/>(本章完)
