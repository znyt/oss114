　　第453章
<br/>　　因为程大雷的缘故，军中兴起一股用斧之风。从底层兵卒到上层指挥官，人人以用斧为荣。军中上层，大部分都是名门之后，打一柄斧头还是不成问题，就算用不惯，也喜欢摆一柄在自己大帐中。
<br/>　　但军中底层士兵，他们大多是苦出身，兵荒马乱，但凡有活下来的办法，也不会去当兵。一柄斧头，至少要几十斤铁，把全部身价拿出来，也未必够打一柄斧头。
<br/>　　拦住程大雷这头目姓章，名叫章螂，已经是一个打了七八年仗的兵油子，七混八混，混了个芝麻绿豆大小的官。在乱军从中，他一眼就相中了程大雷身后的斧头。
<br/>　　于是就想勒索过来，也好在军中露露脸。只不过，他不知道这鬼面斧重达一百单八斤，程大雷之前也只是勉强能使用，何况是章螂。
<br/>　　他一双小眼睛盯着程大雷，骨碌碌转了起来，程大雷心底也有些发毛，这小子该不会认出自己了吧。
<br/>　　希望不要，否则，程大雷虽然不情愿，也只能送他去西天。
<br/>　　忽然，章螂的手狠狠拍在程大雷肩上，道：“好小子，我一眼就看出来你不是一般人，这么重的斧头你用得动？”
<br/>　　“嗯，倒是用得动。”
<br/>　　“果然我没有看错你，你一身好力气，都快赶上我了。”章螂兴奋道：“从现在开始，你跟紧我，随我一起杀敌，砍下贼人脑袋，我把功劳分你一般。”
<br/>　　章螂自家人知道自己事，他本事不如何，又贪生怕死，虽然今天大家都在抢功劳，可真和他们抢，自己也抢不过别人。
<br/>　　眼前这『牛三斤』看上去憨憨的，傻傻的，一副好糊弄的样子。自己就忽悠他多杀几个人，一颗脑袋就能换不少功劳。
<br/>　　程大雷点头不止：“大人，你对我太好了。”
<br/>　　“好啦，好啦，你不用这么感动，我这个人一向是很仗义的。”
<br/>　　程大雷跟在章螂身后，他提着大斧头，章螂握着一杆长矛。看着他手中兵器，程大雷就知道，他在军中混的，非常不怎么样。
<br/>　　程大雷随着大军涌入了杀虎口，而杀虎口中，已经是人满为患。程大雷眼睛滴溜溜乱转，查看杀虎口中的情况。
<br/>　　然后，他瞧见了林少羽。
<br/>　　林少羽现在的情况非常不妙，他带着残兵败将，聚在义字旗下。此刻横枪立马，一人独战帝国军三员大将。
<br/>　　迈入绝世后，林少羽的实力今非昔比，负伤之躯，以一敌三，竟然还撑得下去。但委实撑得很苦，已经是风中残烛，走投无路。
<br/>　　“林少羽，现在全天下都要杀你，你还要撑到何时，若你有种，干脆给自己一个痛快？”李行哉在马上大喝。
<br/>　　程大雷瞥了他一眼，心中恨得咬牙切齿。这李行哉当真是翻脸不认人，知道林少羽是我兄弟，还逼着他自杀。
<br/>　　当然，程大雷也明白，李行哉与林少羽之间的仇恨是不可调和的，各自谁都要杀了对方。
<br/>　　李行哉骑马在阵前，忽然心有所感，回头向程大雷方向望了一眼。程大雷忙低下头去，将身子隐在章螂身后。
<br/>　　夜半三更，用火把驱逐黑暗，李行哉身后乌压压一片，其实他也看不清楚。但他心中隐隐有种感觉，程大雷已经到了。
<br/>　　李行哉一直想逼死林少羽，自然是因为双方之间的仇恨，另外一点原因，就是知道，林少羽不死，程大雷一定会来。
<br/>　　可程大雷救不了林少羽，来了也是送死。只有林少羽死了，程大雷才能断了这念头。
<br/>　　怕此时此刻，程大雷没来，已在来的路上。
<br/>　　李行哉昂起头，眼中杀意弥漫。
<br/>　　“来人，与我取他人头！”
<br/>　　从相府大军中，冲出两员大将，在将军府阵中，也同时杀出两人，此刻一共七人，围着林少羽激战。
<br/>　　现在谁都知道，杀了林少羽是大功一件，所以七人都使出了看家的本领。如此一来，林少羽压力倍增。
<br/>　　能有资格上阵与林少羽交手的，自然是军中好手，而自从打出长安城后，林少羽早已不是全盛之时。
<br/>　　七个人都想要他的命，一招比一招凶险。而林少羽也想杀人，他想要在临死前多要几条命。然而现在的情况是，他可能一个人都杀不死。
<br/>　　人力有时而穷，自己眼前的路，的确已走到尽头。
<br/>　　一枪向林少羽扎来，林少羽回枪拨开，如果是从前，他能顺势一枪要了对方性命。然而，此刻他已没有这样的力气，只能被动防御。
<br/>　　铁枪，钢刀，狼牙棒，镔铁棍……七种兵器同时攻向他，他挡得住一件，挡不住第二件，右臂，左肩，同时种招。
<br/>　　帝国军同时发出欢呼。
<br/>　　再进攻，就是要林少羽性命。
<br/>　　李行哉眼睛眯在一起，心中想的是：早点结束吧，也该结束了。
<br/>　　正在林少羽情况危急时，所有人都看到一道黑影凌空飞起，他身上背着斧头，腰里别着剑。直接将一员大将撞到马下，抡起他的身体，将围攻林少羽的敌人砸飞。
<br/>　　“滚开啦！”
<br/>　　众人耳边只觉得嗡嗡作响，被眼前发生的一幕惊得目瞪口呆。
<br/>　　这一手兔起鹘落，玩得分外漂亮。大家眼睛都盯着马上黑影，心中琢磨着他的来历。
<br/>　　程大雷本不愿如此冒险，从一开始他就琢磨着如何救林少羽脱困，委实想不出什么法子。也不能看着林少羽去死，只能冲了出来。
<br/>　　章螂也随着大家的目光看去，心中还琢磨着，这是谁啊，现在冲出去岂不是送死么。然后他一下认出了程大雷，对方不就是那个看上去有些憨憨傻傻的大汉么。
<br/>　　他之前还在自己身后，刚才是怎么出去的呢。
<br/>　　此刻，已有人呼起程大雷的名字，帝国军各自握紧兵器。起义军这边不知程大雷来历，倒是看清楚了程大雷身后的癞***案，意识到是自己人。
<br/>　　如尉迟离这样的人，皆是紧张万分，虽然这场乱局，程大雷一直未出现，但从未有一家势力，放松过对程大雷的警惕。
<br/>　　李行哉骑在马上，轻轻的叹了口气。
<br/>　　“该来的，果然是来了。”
<br/>(本章完)
