　　第337章
<br/>　　占领琴川关后，程大雷面前千头万绪，一团乱麻。很多事情等着他去处理，案前摆满事情，现在程大雷每天睡觉的时间还不足四个小时。
<br/>　　睡觉的时候还得防备着别人来暗杀，门前三班人站岗，就这样程大雷每天还要捧着剑睡觉。
<br/>　　琴川关虽然只有一地，但辐射范围是广的。十八家头领，附近的村镇……林林总总算下来，程大雷的实际控制范围大概有八百里。
<br/>　　以前这些都是由莫鸣米控制，自有一套规矩在。但如今程大雷取莫鸣米而代之，有些规矩和秩序就要发生改变。
<br/>　　城内的战俘要安抚，城外的十八家头领要控制，流窜四地的莫鸣米残党也要处理……不说其他，现在琴川关的人口已超过一万人，这一万人的吃喝拉撒稍微有点问题，都会酿成大麻烦。
<br/>　　程大雷真的已有些忙不过来了，山寨都是一群大老粗，也没人能帮他。当然，莫鸣米原先留下的一套班子还是能用的，可程大雷也不如何信得过他们，目前这些人还在审核考察阶段。如今，能帮程大雷的竟然只有李行哉、李婉儿两个，可他们也是初出茅庐，很多事也是边学边做。
<br/>　　程大雷决定通过点将台召唤一些人才，目前山寨能打的人程大雷是不缺的，如关鱼、赵子龙、秦蛮这些人都有彪悍武力，对于他们在政治军事上的培养也在进行，但现在还没到能用的阶段。
<br/>　　程大雷这次的目的是召唤一些文职人才，在经济、政治、管理上有出色表现的人。
<br/>　　每一万可以点将一次，每十万连抽十一次，必得一名优秀人才。每百万可兑换一名顶级人才，每千万可兑换一名绝世人才。
<br/>　　这次攻打琴川关，程大雷又获得大量恐惧值，但还有留一些在系统商店购买物资用，马上要开四级本，谁知道系统会开出什么好东西。
<br/>　　所以程大雷这次重点放在十一连抽上，等级太高的未必会听你的，真如果兑出司马懿这种程度的人，他反手把你给篡了也是麻烦不是。
<br/>　　这一日，程大雷沐浴焚香，换了一套干干净净的衣服，恭恭敬敬朝天上了三炷香。
<br/>　　“天灵灵地灵灵，谁也没有系统灵，系统大大你开开天恩，千万不要再让我抽到水货。”
<br/>　　默默祷告一番，程大雷在系统界面选择了点将。
<br/>　　脑海中点将台的图像亮起，有篝火点燃，隐隐似乎有天上的星光落在点将台上。
<br/>　　嘟，获得糟糕游侠……
<br/>　　嘟，获得普通山贼……
<br/>　　嘟，获得普通战士……
<br/>　　嘟……
<br/>　　嘟，获得优秀武将……
<br/>　　……
<br/>　　啧啧，第一把手气有些不顺，程大雷需要的人一个也没有。来的都是些游侠、山贼、樵夫、武将，这都不是程大雷需要的。
<br/>　　程大雷又洗了洗手，希望这次运气会好些。
<br/>　　嘟，获得优秀厨师……
<br/>　　嘟、获得普通医者……
<br/>　　嘟、获得糟糕铁匠……
<br/>　　嘟、获得优秀商人司徒星
<br/>　　……
<br/>　　这一次，要比上次幸运很多，十一次抽取，有五个生活职业，其中还有两个优秀等级的厨师和商人，程大雷把他们的名字记在纸上，等他们投奔过来时，好给他们安排工作。
<br/>　　程大雷一次次抽取，下面的过程可以说不好不坏，有程大雷所需要的，但没有出奇的。
<br/>　　嘟，获得优秀军师吴用。
<br/>　　到后来程大雷已经有些麻木，忽然收到的系统提示令他瞬间来了精神。
<br/>　　吴用，是那个智多星吴用么，和林冲是一个路数来的。可如果真的是他，等级当不会这么低。
<br/>　　程大雷不是很喜欢这个人，从他的名字看就知道，这个人其实没多大用处。梁山水泊有一多半是被他祸害了，而且这个人本事不如何高，狠主意层出不穷，想想有多少忠臣良将被他拉下水。
<br/>　　该不会也是个和徐神机一样的毒奶选手吧。
<br/>　　程大雷这次花费了一千万恐惧值，总共抽了十次，如果直接兑换，已经可以换一名绝世人物了。只不过收获也算不小，达到优秀等级的生活职业足有五十人，包括：军师、书生、厨师、铁匠、医者、商人等等。
<br/>　　过了两三日，这些人都到了，程大雷把他们聚在将军府，亲自给他们训话。
<br/>　　这些人的忠诚是不成问题的，等级达不到，还不知道什么叫背叛。程大雷三说两说，他们就佩服的五体投地。
<br/>　　“你们里面有个叫吴用的？”
<br/>　　一个穷儒站了出来，病殃殃的模样，倒是和刘悲的气质相仿。
<br/>　　“大当家知道我？”
<br/>　　“嗯呐，早就听说啦。”
<br/>　　程大雷注意力集中在对方身上，阅读着此人的信息。
<br/>　　姓名：吴用（初出茅庐的优秀军师）
<br/>　　年龄：29
<br/>　　技能：毒计
<br/>　　隐藏属性：无
<br/>　　原来是年纪还小，看来还有成长空间。只是不知道这人真正本事怎么样，千万不要是个水货。
<br/>　　“我出个问题考考你，看你水平怎么样？”
<br/>　　“大当家请赐教。”
<br/>　　这人倒是很有礼貌，程大雷想了想，道：
<br/>　　“一个笼子里装着鸡和兔，往上看一共有八个头，往下看一共有二十二条腿，问鸡和兔分别有多少只？”
<br/>　　吴用听到后，做出沉思模样，其他人也下意识思索着。
<br/>　　程大雷挥挥手指着其他人：“你们也琢磨琢磨，答得上来的本寨主有赏。”
<br/>　　所有人一起思考，足足过了一炷香时间，程大雷有些头疼。这个世界数学普及率就这么低么，简单的鸡兔同笼问题都要过这么半晌。
<br/>　　“我知道了。”人群中有一人举起了手。
<br/>　　程大雷点点头：“你叫什么名字？”
<br/>　　“禀大当家，我叫司徒星，原先是个客商，这次特意来投奔大当家。”
<br/>　　“很好，你就留下吧，以后有用得着的地步。”
<br/>　　程大雷目光看向吴用：“你呢？”
<br/>　　吴用轻轻点点头：“在下早已算到了。”
<br/>　　程大雷愈发头疼，看着自鸣得意的模样，怎么看怎么感觉与徐神机有几分神似。
<br/>　　这种小学三年级的数学题，又必要这么得意么。
<br/>　　程大雷很想登高大呼一声：非战之罪，是天不助我呐。
<br/>　　
<br/>(本章完)
