　　第485章
<br/>　　李乐天笑了笑，道：“你打算要什么？”
<br/>　　纵然他笑得一如从前随意而懒散，但程大雷仍能察觉他眼底的沧桑和苦涩。在样一场乱局中，谁也无法全身而退，李行哉如此，程大雷又何尝不是如此。
<br/>　　“林少羽死了。”程大雷淡淡道。
<br/>　　“但我父皇也死了，长安城也死了很多人，并州也死了很多人。”李乐天看着程大雷的眼睛：“林少羽该死。”
<br/>　　房间内的气氛陡然沉重下来，程大雷眼皮剧烈的跳动了两下。半晌，他轻轻舒出一口气：“是呐，他该死，但很多该死的人还活着。”
<br/>　　“你杀不干净的。”李行哉道：“就算把他们都杀干净，这乱世也不会变得更好。”
<br/>　　程大雷沉默以对，刘悲和徐神机都离开了房间，只有沉默相对的李行哉和程大雷。
<br/>　　“春秋无义战。”
<br/>　　李行哉缓缓吐出五个字，程大雷轻轻点了点头，认同了李行哉的看法。
<br/>　　春秋无义战，乱世也无义战。就算林少羽举着正义的大旗，也未必就是正义的。而程大雷，根本就不相信世上有正义这种东西，无非是你杀我，我杀你，杀来杀去，最后把人都杀光了，乱世便结束了。然后周而复始，开始下一个循环。
<br/>　　“再打下去，无非是双方互相死人，你的蛤蟆寨也无法避免。”李行哉道：“就这样结束吧，你开出个条件，城池，爵位，为了让你退兵，帝国可是准备了极好的价码。”
<br/>　　“如果我真要这些，我也不至于走到今日这一幕。”程大雷笑了笑，眼底的苦涩去了很多。
<br/>　　“我明白，其实很多人都在想，你究竟要干什么，连我也没想明白。”李行哉。
<br/>　　“好了。”程大雷道：“既然你来了，我不能给你面子。你说退兵，就退兵，咱给你这个面子。城池、爵位我一概不要。按你说的，林少羽该死，我也觉得他该死，他被李乐天杀了，杀父之仇，我也不好说什么。现在，我要林少羽的遗体，这个要求不过分吧。”
<br/>　　“不过分。”
<br/>　　李行哉点点头。林少羽的死，为帝国不少人都换来了功勋，他的武器、盔甲、战马，尸体……都可以换来功勋，只不过还未回到长安，还没有来得及论功行赏。
<br/>　　现在程大雷要讨回去，对帝国军来说自然是极大的屈辱，可到现在这种情况，也谈不上什么屈辱不屈辱。
<br/>　　事情比李行哉想得要顺利，但他也没觉得这件事有太大的困难。程大雷终究不是杀魔，要将帝国军杀光了为林少羽复仇。
<br/>　　他只是因为林少羽死了不开心而已，至于该做什么，不该做什么，其实在这个乱世中，谁也不明白。
<br/>　　“好啦，李行哉，我答应你了。但你要知道，不是你说服了我，而是我想被你说服。”程大雷看着李行哉：“你最近混得怎样？”
<br/>　　“不太好。”李行哉指着自己的左腿：“瞧瞧你差点把我一条腿废了。”
<br/>　　“不是能接上么，少上几次战场，说不得还能保条命。”程大雷道：“你自己好好保重，这乱世刚刚开始，但总是要结束的。我和你们一样，都希望这乱世能早些结束，我更希望，未来能结束这乱世的人是你。”
<br/>　　“借你吉言。”李行哉道：“好啦，我要走了，诸事不由身，日后若有机会，我们再见面吧。”
<br/>　　“日后再见面……”程大雷笑了笑，把后面的话压在心底：真不希望，日后再见面，我们会成为敌人。
<br/>　　黄沙城头，落日已经快要落山，夕阳给古城渡上一层金黄，草原上有新绿从枯草中冒出个头。
<br/>　　但这个时候，还没人下城，都在等着对面蛤蟆军大营传来的消息。
<br/>　　一探子快马从城外而来，疾奔上了城头。
<br/>　　“报！”
<br/>　　“怎样了，六殿下出来没有？”李乐天立刻问。
<br/>　　“殿下已经离开敌营，正在返回。”探子气喘吁吁道。
<br/>　　李乐天长长的松了口气，虽然外人将李行哉视为废物皇子，可却是明帝最看中的人。而李乐天对自己这个六弟，除了嫉妒和羡慕，其实也十分看重。
<br/>　　知道他平安的消息，李乐天胸口的石头放了下来。他立刻派出人去，将李行哉接过来。自己与尉迟离等人亲自在城门迎接。
<br/>　　只见一乘小轿缓缓从远方而来，在空旷的草原上显得如此渺小，但随着它的靠近，在众人的视线中却越来越大。
<br/>　　“六弟……”
<br/>　　“六殿下。”
<br/>　　轿帘刚打开，诸人便围了上去，现在他一个人，系着无数人的心。
<br/>　　“谈妥了。”李行哉腿伤不能下轿，只能坐在轿内说话：“他答应退兵。”
<br/>　　这一瞬，不知多少人都出了口气，一时间竟没有人说话。
<br/>　　“条件呢，他提出了什么条件？”尉迟离问。
<br/>　　“这件事先不着急，我没进城再谈。”李乐天。
<br/>　　“对对对。”崔相道：“六殿下以一人之力，拯救数万士兵的安危，这里每一条命都是殿下所救。”
<br/>　　尉迟离直咬牙，心中想这崔相和李行哉实实在在可恨。本来表现堪忧的李行哉，却借着这件事，重新登上了挑战帝位的擂台。
<br/>　　程大雷也真是的，为何不一斧头劈砍了他，如此，岂不是给自己省了不少麻烦。
<br/>　　现在，李行哉又收获了不少人心，他虽然心中不服，但也是满脸赞许道：“六殿下勇哉，实在令老夫佩服。这里风寒露重，殿下身上有伤，咱们还是回城再说。”
<br/>　　城主府的大厅内灯火通明，李行哉罕见的坐在了李乐天下手的位置。在大家注视的目光中，李行哉说出了程大雷开出的条件。
<br/>　　“难道只要林少羽的遗物，他就可以退兵？”尉迟离皱起眉头：“这件事会不会有诈？”
<br/>　　不怪尉迟离不相信，实在程大雷开出的条件太简单了些，简单到按正常思维无法相信。
<br/>　　“我们倒是不可轻信了贼人，山贼么，很难有什么诚信。”尉迟离的手下附和道。
<br/>　　李行哉看向李乐天，李乐天轻咳一声：“按他所提的条件准备吧，最好，今晚就能完成交接。”
<br/>(本章完)
