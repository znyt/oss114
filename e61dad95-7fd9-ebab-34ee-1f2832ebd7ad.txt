　　萧逸傲然的话语中，又带着半分戏谑与玩味。
<br/>　　毒袍闻言，身躯猛地一颤。
<br/>　　对，在他所知晓的信息中，紫炎易霄，是个控火天才，是个控火妖孽。
<br/>　　但同时，紫炎易霄，可是个传闻中游走于各大险地、各种九死一生之境，仍旧安然无恙的强者。
<br/>　　这样的妖孽，这样的强者，怎可能会只有一种手段。
<br/>　　这样的妖孽，或许任何在外人眼中视为惊讶的事情，放到他身上，都会显得再普通不过。
<br/>　　“是我小看了你。”毒袍的脸色，陡然难看到极点，直视萧逸。
<br/>　　萧逸淡漠一笑，双脚，稳稳的踏在这头巨大血龙之上，身上气势，滔天惊人。
<br/>　　“现今，六龙六煞，不算龙啸，你我各执两条。”
<br/>　　“你还有胜算吗？”
<br/>　　不错，这两条巨大的血龙、风龙，早已在萧逸的掌控之下。
<br/>　　这也是为什么毒袍之前操控二龙，攻击龙啸与东方铁云时，萧逸从远方而至，一声暴喝，便已止下了这两条巨龙。
<br/>　　甚至于，萧逸此刻稳稳地站在二龙头颅之上。
<br/>　　“易霄分殿主，有两条龙形煞气被你掌控了？”一旁，东方铁云脸色霎时惊喜。
<br/>　　萧逸点了点头。
<br/>　　倒是一旁龙啸，脸色愈发难看，“这是我龙家的龙形煞气。”
<br/>　　“易霄，叫掌控交回于本域主，本域主会亲自结束今日的邪修祸患。”
<br/>　　萧逸摇了摇头。
<br/>　　龙啸脸色一冷，“易霄，你想做什么？”
<br/>　　萧逸淡漠道，“邪道封天阵下，你暂时掌控不了。”
<br/>　　萧逸没有说谎。
<br/>　　六龙六煞，本是龙家的东西。
<br/>　　传闻，六龙地域这边，本就有地域之利，应运而生地在地域六方诞生了六条龙形煞气。
<br/>　　当然，煞气初生时，尽皆脆弱而渺小。
<br/>　　龙家先祖，在此定居，创立家族，且加以培养，致使六龙煞气愈发强大。
<br/>　　百万年来，六龙煞气在地域六方孕育，由龙家倾力培养，方有今日庞大无比，长逾十万丈。
<br/>　　且每每调动，便整个地域震颤不堪，威势滔天。
<br/>　　六龙煞气，本就是受龙家历代家主掌控。
<br/>　　今日，本该受龙家掌控的六龙煞气，竟有其四在毒袍手中。
<br/>　　萧逸这一月时间，都在地域之外养伤，不太清楚这边的事。
<br/>　　但他若没有猜错的话，必然是龙啸被什么事情拖住了，且大意下，眼睁睁被毒袍等人布下邪道封天阵。
<br/>　　邪道封天阵，邪君府最强的上古大阵之一。
<br/>　　甚至可与修罗殿的修罗锁天阵，黑魔殿的黑魔炼妖大阵相媲美。
<br/>　　之前萧逸在了六龙地域之外时，明显看到，这个大阵，竟然撼动了整个地域。
<br/>　　证明，布置此阵的材料，必然极其大量，而且皆是重宝。
<br/>　　再加上之前三长老所说的，龙啸放开龙家宝库，任他索取重宝。
<br/>　　那么，情况不言而喻。
<br/>　　再加上毒袍、虎护法，还有十万邪修维持大阵。
<br/>　　此大阵，方才有了惊天之力。
<br/>　　最后，便是六龙煞气中，有一条邪煞。
<br/>　　邪煞，乃邪道之煞，邪君府中人，本就善于控制。
<br/>　　邪道封天阵此上古大阵。
<br/>　　无数重宝和材料下所布置此大阵。
<br/>　　外加毒袍、虎护法、十万邪修。
<br/>　　最后，六龙煞气之一的邪煞作为契机。
<br/>　　多重配合下，便有了毒袍直接抢夺了邪煞的控制权。
<br/>　　邪煞之下，再配合大阵，压制另外5条煞气。
<br/>　　按理说，毒袍应该能将6条煞气的控制权完全夺去。
<br/>　　可龙啸，显然也非泛泛之辈；龙家功法，也极强。
<br/>　　故，龙啸堪堪抢回了两条龙形煞气的控制权。
<br/>　　这，便有了之前一开始的，六条煞气，毒袍占其四，龙啸仅占其二。
<br/>　　若萧逸现今将血煞、和风煞的掌控权交出；恐怕龙啸还未能掌控，就已经再次被毒袍给夺回了去。
<br/>　　至于他为什么能掌控血煞和风煞。
<br/>　　原因很简单，他体内那颗血丹，顶住了毒袍的邪煞压制。
<br/>　　“给本护法回来。”毒袍猛地暴喝一声，双手齐出。
<br/>　　双手凌空吸摄，脚下那条阴寒邪煞，光芒涌动。
<br/>　　萧逸脚下血煞，震了震，轻移半分。
<br/>　　“哼。”萧逸冷哼一声，脚步一踏。
<br/>　　脚下血龙，瞬间恢复了正常。
<br/>　　萧逸内视了一眼，明显可以看到，体内那颗血丹，同样在光芒熠熠。
<br/>　　“这条血煞的控制权，你夺不回去了。”萧逸冷笑一声。
<br/>　　毒袍能操控六煞，说白了，便是邪道封天阵，控制了邪煞；而他，又通过邪煞，压制了其余五条煞气。
<br/>　　刚才，毒袍便打算操控邪煞，压制血煞，从而夺回操控权。。
<br/>　　只是，邪煞对血煞的压制，显然远不如萧逸体内血丹对血煞的操控力。
<br/>　　他的血丹，本就在上次剑帝碑外，杀了大量天下剑修下，沾满了大量剑修的血气之力，早就成长到相当可怕的地步。
<br/>　　毒袍的大阵之力，本就要压制着龙啸的操控。
<br/>　　他单凭邪煞压制，如何能从萧逸的血丹压制下夺回血煞？
<br/>　　再者，萧逸还修了血刹魔经，单凭对血煞等血气力量的掌控，绝对超过了龙家的功法所对血煞的掌控。
<br/>　　龙啸，尚且能在毒袍的大阵压制下，夺回两条煞气的掌控权。
<br/>　　何况他萧逸？
<br/>　　至于风煞，那就更简单了。
<br/>　　毒袍能用邪煞压制其它五煞，以得到掌控权。
<br/>　　萧逸，同样能通过血煞，压制风煞，获得掌控权。
<br/>　　龙啸、毒袍、萧逸，三人其实都很清楚，胜负，根本就掌控在对六龙煞气的操控下。
<br/>　　在六龙地域，得六龙煞气者，便是无敌。
<br/>　　自然的，三人手段齐出，各执掌控权。
<br/>　　此刻三人未动手，但暗自的交锋、拉扯，早已激烈异常。
<br/>　　毒袍，蓦地收回了双手，停止了吸摄。
<br/>　　他全力之下，根本再也撼动不了血煞和风煞的控制权。
<br/>　　萧逸淡漠一笑，“那些暗地里的交锋，想必无需再继续了。”
<br/>　　“毒袍，你我二人如今各执两条煞气，仍旧如一月前那般，公平一战。”
<br/>　　“不过，今日我需得分出胜负，也不会再让你逃去。”
<br/>　　冰冷的话语，瞬间落下。
<br/>　　萧逸脚下两条巨龙，瞬间而动。
