　　萧逸迫不及待地接过两本书籍，看了起来。
<br/>　　二长老一愣，道，“臭小子，急什么。”
<br/>　　萧逸没有理会，自顾翻开了第一本书籍。
<br/>　　“噬火百刃，玄阶低级武技，炼至大成，百道火刃如臂指使，刀山火海，敌人无招架之力。”
<br/>　　书籍首页上的介绍，立马吸引了他的目光。
<br/>　　二长老低声笑骂道，“别人说我是武痴，你反倒比我还痴迷。而且，此乃我萧家最高深的武技，极难练成，你现在看也难有收获....”
<br/>　　“嘘。”一旁的大长老忽然做了个噤声的手势，低声道，“别打扰他。”
<br/>　　气氛忽然变得静谧，鸦雀无声，三位长老静静地看着萧逸。周遭只剩下萧逸翻书的细微声响。
<br/>　　半晌，当萧逸合拢书籍时，双眼精光爆射，蓦然地伸出了右手，手上一阵真气涌动。
<br/>　　真气渐渐浓缩并凝集，一道散发着炙热高温的火刃凭空而现。
<br/>　　“去。”萧逸大喝一声，火刃脱手而出。
<br/>　　火刃最后打到地面上，先是将地面破开一道深深的裂痕，随后轰然爆炸，炸出一个大坑。
<br/>　　“这...”三位长老同时一惊，“不过短短一会儿，竟领悟了噬火百刃，更成功凝聚了一道火刃。”
<br/>　　武技，是炎龙大陆最珍贵的东西之一。
<br/>　　同时，它也是无数武者梦寐以求的存在，比之丹药更具吸引性。
<br/>　　同等级的武者中，拥有武技的一方，绝对有着碾压性的实力。
<br/>　　武技，从高到低共有四个阶级，天、地、玄、黄。
<br/>　　而每个阶级，又分低级、中级、高级和巅峰四个等级。
<br/>　　每个等级间的差距，非常大。
<br/>　　越是高级的武技，威力就越是恐怖。
<br/>　　就算是最低等的黄阶武技，都能让武者发挥出数倍于自身的战力。
<br/>　　而玄阶武技，则能让武者轻松越级战斗。
<br/>　　至于地阶、天阶武技，其威力就更加强大了。
<br/>　　传言，一本天阶武技，足以让一个后天境武者斩杀洞玄境武者，无视武道修为间的境界差距。
<br/>　　当然了，这只是传言，谁也没有见过天阶武技，那几乎是属于传说中的东西。
<br/>　　而地阶武技，传闻只有那些大宗派和超级势力中才存在。
<br/>　　整个紫云城，最强大的武技，只有玄阶低级，而且还是三大家族才拥有。
<br/>　　至于那些小家族，或者独行武者，能拥有一本黄阶武技已经很不错了。
<br/>　　紫云城内，曾经有过著名的商行拍卖一本黄阶巅峰武技，当时的起拍价是五万两，但拍卖者络绎不绝，甚至是哄抢，最终竟以八万两的价格被卖出。
<br/>　　黄阶巅峰武技尚且如此珍贵，三大家族各自拥有的唯一一本玄阶低级武技，其珍贵程度不言而喻。
<br/>　　言归正传，三位长老愣愣地看着萧逸，脸色中尽是不可置信。
<br/>　　“虽然只凝聚出一道火刃，未到炉火纯青的大成之境，但也代表着他已经初步掌握噬火百刃。”大长老赞赏地点了点头。
<br/>　　“这个臭小子是个变态吗，老夫当年足足修炼了十天十夜才凝聚出第一道火刃，他倒好，才刚把武技看完，这就领悟了。”二长老看向萧逸的目光仿佛在看一个怪物。
<br/>　　三长老则是笑笑，不语，笑容中尽是自豪。
<br/>　　“好了。”大长老忽然出言，打断了萧逸观看下一本武技的念头。
<br/>　　“萧逸，噬火百刃是我萧家最重要的武技，不可外传，今日记下武技上的信息后，明日归还，记清楚了吗？”大长老沉声说道。
<br/>　　“嗯。”萧逸点点头。
<br/>　　大长老继续说道，“至于下一本武技，这是家主当年失踪前留下的武技，整个萧家暂时只有我们三位长老看过。现在交于你，并由你决定是否继续交给家族保管，或者给其他族人修习。”
<br/>　　萧逸陷入了沉思。
<br/>　　“不用急着做出决定。”大长老道，“这是你的东西，如何决定我们都没有意见。另外，前往陨星山脉之事，我劝你同样不要着急，那里远比你想象中的要危险。”
<br/>　　“家族规定，所有达到条件的子弟都必须派出去历练。但这次，我给予你特权，你可以选择过几年再去，或者不去也没有问题。”
<br/>　　“你的天赋，是老夫生平从未所见，假以时日，必将成为名震紫云城的强者。老夫更希望看到你日后成长起来，而不是现在出去冒险历练。当然了，该如何选择，还是由你自己决定。”
<br/>　　说罢，大长老摆了摆手，道，“好了，我们还有其他要事商议，如果没其他事的话，退下吧。”
<br/>　　萧逸点点头，转身缓缓退了出去。
<br/>　　萧逸离开，却不知，府邸内，三位长老都忽然同时露出了意味深长的满意笑容。
<br/>　　“这小子，很好。”大长老笑着说了一句。
<br/>　　“嗯。”二长老点点头，道，“论武道天赋和炼药资质，都比你我要强得多。”
<br/>　　“我说的不是这个。”大长老摇摇头。
<br/>　　“哦？”
<br/>　　“我说的是他的心智。”大长老沉声道，“从家族大比开始，被数位长老的针对，一直到如今的谣言，他的表现都让我相当满意。”
<br/>　　“他很成熟，面对各种针对没有任何慌张。特别是对各种问题的思考和应对，几近完美。而在这成熟中，他又有着一股倔强和傲气。”
<br/>　　大长老边说着，白了二长老一眼，道，“对比他的成熟，在我眼中，你反倒像是个幼稚的少年。”
<br/>　　“切。”二长老撇撇嘴，懒得反驳。
<br/>　　大长老笑了笑，道，“这个世界不乏天才，但大多天才，都在他们还未强大起来前便出于各种原因夭折了。至于萧逸，我几乎可以肯定，以他的心智，绝对能成长起来。”
<br/>　　“不出十年，他将超过我们这些老家伙，成为更强的武者。”
<br/>　　“只是。”大长老忽然皱起了眉头，道，“让我想不明白的是，他一个连萧家大门都没出过几次的少年，怎么会有如此深沉的心思，深思熟虑，像个饱经风霜的人。”
<br/>　　一个少年自然不会有这样的心智。但一个穿越过来的，常年游走于生死危机的杀手之王却有。
<br/>　　一旁的三长老出言道，“或许是，这些年来他在家族中的遭遇让他成长得这么快吧。”
<br/>　　大长老的脸色忽然一凛，叹了口气，道，“唉，这些年，苦了他了。二长老，待会儿你去五长老那敲打一番，让他收起那些小动作。”
<br/>　　“好。”二长老点头道，“萧逸是我们萧家有史以来难见的天才，自然不能再让他受委屈了。”
<br/>　　“三长老。”大长老继续吩咐道，“待会儿，你也去家族各堂拿些东西，丹药材料银两什么的，该拿便拿，确实该好好补偿萧逸。”
<br/>　　“是。”三长老笑着应答。
<br/>　　吩咐完，大长老起身，准备离开。他身为家族唯一的三品炼药师，平日里非常忙。
<br/>　　“慢着。”二长老忽然叫住了他。
<br/>　　“有事？”大长老挑了挑眉头。
<br/>　　二长老道，“萧逸这孩子应有的待遇、补偿等等，我们都重新给他。那么，这些年瞒着他的事，是否也...”
<br/>　　“这...”大长老陷入了沉默，不知如何回答。
<br/>　　二长老继续道，“家主当年并非忽然失踪，而是自己离开的。萧逸作为他的儿子，我觉得他有权知道。”
<br/>　　半晌，大长老看向三长老，道，“三长老，当年家主离开前，最是信任你，更是将年幼的萧逸托付于你。我觉得，到底要不要告诉萧逸，让你来决定比较好。”
<br/>　　三长老摇了摇头，忽然一笑，道，“当年的事，我不会告诉逸儿的。如果有可能，我会瞒他一辈子。”
<br/>　　“你...”二长老皱眉道，“你这样对他很不公平。”
<br/>　　三长老仍旧摇摇头，认真地说道，“我只是为了逸儿好。家主当年为什么要离开，又去了哪里，为什么这么久不回来，这一切的一切，我们又知道多少呢？”
<br/>　　“再者，如果逸儿日后有本事，该知道的，就算我不说，他自己也会知道。若他没本事，我就算说了，也只不过让他心里多一道负担。”
<br/>　　二长老还想说些什么，大长老打断道，“算了，三长老对萧逸视如己出，他对萧逸的付出，远超我们。萧逸的事，他比我们更有权利去决定。”
<br/>　　......
<br/>　　萧逸早已回到家中，对三位长老的对话一无所知。
