　　此时，暗道内又是另外一幅光景，这是一个密室，虽然空间不大，但是家具却也一应俱全。
<br/>　　而坐在一张桌子后面，正持笔书写着什么的矮胖中年人，正是任家的家主，任发。
<br/>　　似乎是感应到上面所发出的声音，任发停下了手中的动作，抬头看了一眼，正好看见秃头老人那张鬼气森森的脸庞。
<br/>　　“钱老，那批货运出去了吗？”放下毛笔，任发随手拂了拂袖袍，开口问道。
<br/>　　此时的任发，一脸严肃，却不像昨日在茶楼见面时那般笑呵呵的了。
<br/>　　“老爷，货已经找人送了，估计后天就可以进行交接。”秃头老人躬着身子，一脸恭敬的回答道。
<br/>　　点了点头，似乎是想起了什么，任发继续问道：“对了，九叔那个老家伙，没什么异常吧。”
<br/>　　说起九叔的时候，任发眼中带着凝重，如果那个老家伙发觉了这件事情，那可就大事不妙了。
<br/>　　“老爷，九叔那里倒是没什么异常，但是他新收的那个徒弟，今天早上来府上了，说是要找小姐。”被称为钱老的秃头男子，将之前发生的事情，原原本本叙述了一遍。
<br/>　　而他的话语，也让任发皱起眉头，背负双手在原地走了几步，嘀咕道：“九叔的徒弟，来找婷婷？”
<br/>　　看了钱老一眼，心中似乎在思考着什么，良久，有些迟疑的开口：“这件事情，我知道了，那穷小子，应该只是想要追求婷婷，暂时不要管他，货比较重要。”
<br/>　　“婷婷那边，也不要告诉她，过段时间我会找个机会把她重新送回省城，这种事情就不要让她参与进去了。”
<br/>　　说罢，任发将桌上的油灯吹灭，招呼了一声钱老后，两人双双走了上去。
<br/>　　……
<br/>　　已经在房梁上趴了很久，但是刘哔依旧没有失去耐心，他眼睁睁看到秃头老人下去，那么对方一定会上来的。
<br/>　　果不其然，一炷香的功夫，暗道的入口再次打开，随之而出的，还有任老爷那并不清晰的声音。
<br/>　　“不……诉她，过段时间……机会……省城，这种事情就不要让她参与进去了。”
<br/>　　暗道的开启需要一点时间，任发的话也有些前言不搭后语，刘哔皱着眉头，猜测着这段话里透露出来的意思。
<br/>　　任发笨拙的身躯从暗道口走了出来，随之一起的，还有之前下去的秃头老人，但是两人并未在大厅内过多停留，径直从大门走了出去。
<br/>　　又是等待了一会，确认两人已经走远，刘哔方才从房梁上跃了下来，重新开启了书架上的机关。
<br/>　　他知道时间不多，故此没有任何多余的动作，等暗道口开启之后，就急匆匆的冲了进去，他想里面，或许会给他带来一些线索。
<br/>　　密道内没有一丝光亮，黑洞洞的，显得有些阴森。
<br/>　　但是黑暗并未对刘哔产生什么影响，周遭的情况依旧清晰可见，他顺着木质台阶向下走着，没有任何迟疑。
<br/>　　脚下的台阶已经有些腐朽，脚踩上去发出‘吱嘎吱嘎’的响声，因为空间有限，他也只能弓着身子，有些憋屈。
<br/>　　片刻，刘哔走到了尽头，看见了一扇上锁的木门，不禁挑了一下眉头，直接将门上的铁锁整个拧下。
<br/>　　他没有钥匙，也不会开锁，想要进入只能用这种强制的手法。
<br/>　　铁锁被破坏时发出一阵令人牙酸的声音，在这寂静的密道里格外明显，刘哔推开木门，眼前豁然开朗。
<br/>　　里面的空间并不算小，有床，有书柜，看样子应该是一间卧室，里面没有点灯，从通风口处洒下的些许亮光，稍微驱散了房间里的昏暗。
<br/>　　刘哔环视了一圈，目光被一副巨大的壁画所吸引。
<br/>　　那张壁画约有两米多长，巨大无比，用一个精致的画框裱在墙上，上面绘着一个栩栩如生的紫衣女子。
<br/>　　画中的女子肌肤赛雪，美若天仙，散发出一股清冷出尘的气质，其容貌足以让所有男人为之疯狂。
<br/>　　但吸引他的，是女子脸上的胎记，这块胎记形似恶鬼，覆盖了女子的小半张左脸，看上去触目惊心，显得有些狰狞。
<br/>　　“可惜了这样一张漂亮脸蛋。”轻叹一声，刘哔将视线移开，在房间内仔细翻找着。
<br/>　　片刻的时间，他在书桌上找到了一封还未完成的书信，粗略的查看一遍后，揣进自己的衣兜里。
<br/>　　信上的内容有些奇怪，好像是任发要非法收购某种货物，看样子还很着急，但信没写完，他也不知道具体是怎么回事。
<br/>　　又搜索了一会，没再有什么收获，怕有人发现，刘哔揣着信纸，准备从密道内离开，心中却突然涌上一股危险的感觉。
<br/>　　没有任何犹豫，刘哔双腿发力，闪电般的向旁边躲去，‘砰’的一声，他之前站立的地方，出现了一个大洞。
<br/>　　突如其来的攻击，震的整个密室微微发颤，惊魂未定的向四周看去，却没能找到攻击他的敌人在哪。
<br/>　　刘哔眉头紧皱，全身的肌肉紧绷着，一股淡淡的恐惧感爬上心头。
<br/>　　四周静悄悄的，周围的环境一目了然，有限的空间里，根本不可能隐藏下任何敌人。
<br/>　　这个时候，他想起任家闹鬼的传闻，心中合计，莫不是真的遇见鬼了？
<br/>　　就在刘哔进退维谷之时，突然像是感应到了什么，猛然回身，一拳打向身后的空气，霎时，一声凄厉的嚎叫声传了出来。
<br/>　　身后空荡荡的，没有任何敌人的身影，但他非常确信，之前的攻击，一定打中了什么东西。
<br/>　　拳头通红一片，微微有些发痛，刘哔心中无奈，他又找不到敌人的身影了。
<br/>　　不能这样拖下去，刚才那声嚎叫，怕是要引来任家的人，不想过早的暴露自己，刘哔心中已经开始萌生退意。
<br/>　　但是，那个隐藏在暗处的敌人，似乎并不想他离开一样，就在他刚刚抬脚的时候，突然脖颈一痒，心中顿时警铃大作。
<br/>　　那种感觉，就像是有人在他身后，对着他的脖子呼吸一般，刘哔头皮一麻，反手一拳向后面挥去。
