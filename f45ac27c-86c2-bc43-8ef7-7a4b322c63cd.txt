　　此时，众人头顶上方，一阵嘶吼声传来，声音虽然清楚，可谁都无法分辨这究竟是什么东西。
<br/>　　“打开电梯门，如果在这里发生战斗，电梯会支撑不住的。”刘哔持着黑色棍棒，脸色微微有些阴沉：“从这里摔下去，我们都得死。”
<br/>　　电梯内的液晶显示表上，数字已经跳动到了十层，这样的高度，在场的这些人，恐怕也有他能存活下来了。
<br/>　　这样的结果，可不是他想要看到的。
<br/>　　“打不开啊，电梯还在运行中，无法被强制开启的！”
<br/>　　白小飞自然也清楚事情的严重性，但不管他怎么着急的摁动按钮，电梯的大门依然是没有半点反应。
<br/>　　草，这智商，是猪么！
<br/>　　暗骂一句，刘哔夺过白小飞手中的长刀，直接挥向电梯的铁门。
<br/>　　空间太狭小，他的武器虽强，但此时却不如一柄长刀更加适用了。
<br/>　　“嗤啦……”
<br/>　　一阵令人牙酸的声音响起，那是刘哔手中的长刀在铁门上划过的声音，在他的攻击下，钢铁大门如同豆腐一般，被划开了一道足以让众人通过的口子。
<br/>　　看了看外面的场景，刘哔舒了口气，还好，电梯的上半截正好停在了十层的楼道里，费点力气，倒也能出的去。
<br/>　　总比停在两个楼层中央的电梯井里要好办的多。
<br/>　　“快出去！这电梯支撑不了多长时间了！”
<br/>　　怒吼一声，白小飞探出了头，确认周围没危险之后，率先将小鹿扔了出去。
<br/>　　紧接着，众人连攀带爬的跑出了电梯，还不等众人喘口气，身后的电梯发出一声巨响，顺着电梯井急速滑落下去了。
<br/>　　惊鸿之间，几人都瞥见了一个和巨蜥有几分相似的庞大身影，这个生物背对着楼道，似乎并没有发现他们。
<br/>　　“草，真险，小爷差点就交代在了这里。”
<br/>　　拍了拍胸口，白小飞盯着黑洞洞的电梯口，脸上带着劫后余生的庆幸。
<br/>　　所有人额上都是冷汗直流，目光中透着后怕的神色，刚才在晚一点，他们就要全部交代在这里了。
<br/>　　“走吧，先离开这里，谁知道那个怪物会不会折返回来。”
<br/>　　喘了一口粗气，白小飞缓过神来，带领着大家离开了电梯门前。
<br/>　　因为电梯已经不能使用了，接下来的两层，众人都要在步行通道通过，虽然对大厦并不熟悉，但步行楼梯并不难找。
<br/>　　像这种建筑，布局都是大致差不多的，步行楼道也一定是建立在方便明显，又不失美观的地方。
<br/>　　往大概会设立步行通道的地方寻找了一圈，几分钟的功夫，几人就站在了楼道门前。
<br/>　　楼道的小门紧闭着，黑洞洞的，白小飞拿手机照了一下，也没看见有什么东西。
<br/>　　“打开看看吧，总要打开的，我们不可能一直留在这里。”
<br/>　　似乎是为了安慰自己，白小飞嘀咕了一句，狠狠推开小门，但出乎众人意料的，小门并没上锁，白小飞用力过猛，差点摔了出去。
<br/>　　“想死么？”
<br/>　　一把抓住白小飞的后衣领，刘哔眉头微皱，看着前方布满了整个楼道的虫卵。
<br/>　　大概是那些变异蚊虫留下的，连它们的卵也都变异了，这些虫卵，每一个都有小孩头颅大小，挂满了楼道内的每一个角落。
<br/>　　一阵蠕动的声音响起，透过手机电筒的照射，很清晰能看见里面蠕动的变异蚊虫，白小飞转头看了看刘哔，向征询一下刘哔的意见。
<br/>　　这样的情况，他也不知道应该怎么办了，虫卵这么密集，众人要想要通过，难免会将其触碰到。
<br/>　　如果里面的变异蚊虫出来，和他们发生了战斗，那将会引起连锁反应。
<br/>　　这么多虫卵，一旦全部破壳而出，而且还是在这种地形下，他们根本没有逃出去的机会。
<br/>　　“小心点走吧，只能尽量注意一下，没有别的办法了。”刘哔沉吟一下，有些无奈的说道：“用小飞的话来说，这也是没有办法的事，我们无论如何都要走这条路的。”
<br/>　　叹了口气，白小飞点了点头，主动走到了队伍前方。
<br/>　　是啊，没别的办法了。
<br/>　　“沙沙……”
<br/>　　众人小心翼翼的前进着，所幸在这关键的时刻，并没有发生什么意外，很快众人就爬到了十二层，看到近在咫尺的入口，白小飞微微松了口气。
<br/>　　“小鹿，已经到了，可以回……”
<br/>　　转过身来，白小飞脸上挂着微笑，但话语说到一半的时候，情况突变，一根巨大的尾巴将小门击碎，瞬间卷住白小飞的腰部，向内拖去。
<br/>　　那是一条浅褐色的尾巴，表皮有些粗糙，像极了蜥蜴，但体型却是令人难以想象。
<br/>　　突如其来的变故，让白小飞根本没有半点反抗的余地，只能留下一声越拉越远的惨叫声。
<br/>　　碎铁片溅射进楼道内，打破了附近墙壁上的虫卵，顿时，密集的蛋壳碎裂声响起，这一下，彻底激起了连锁反应。
<br/>　　“草！”
<br/>　　暗骂一声，刘哔将保安小哥和小鹿两人扔出门外，顺手往身后放了一把火。
<br/>　　“轰！”
<br/>　　炙热的火舌升起，但此时他也没时间关心自己这一把火的效果了，外面，还有一个东西等着他去解决。
<br/>　　真不知道这些人是怎么活下来的……
<br/>　　嘀咕了一声，刘哔转身窜出楼道，迎面撞上了一个长相奇异的怪物。
<br/>　　之所以说它奇异，是因为这家伙长了两个脑袋，和人头相似，可偏偏长在了背上，而这家伙的身体，却是和壁虎有些相似。
<br/>　　至于壁虎身躯原本头部的位置，他有些不确定这还算不算是头部了，没有眼睛，没有鼻子，甚至连耳朵都没有。
<br/>　　如果不是还留着一张没有牙齿的嘴巴，与其说是头部，倒不如说，是一个长了嘴巴的屁股，那样要更加形象一些。
<br/>　　“哥哥快看，这里还有，这些我们能饱餐一顿了。”
<br/>　　注意到这边的动静，壁虎怪物背上的左侧头颅舔了舔嘴唇，口中竟是发出了人语。
<br/>　　这个家伙……之前是人类？
<br/>　　居然，还保留着人类的智慧和思维能力么？！
<br/>　　这个世界的丧尸，可真是让他三观尽毁呢。
